# Ericsson Technology Review: CPO External Laser Source And Thermal Stress

Key takeaways:

- Ericsson frames external laser sources as a way to improve reliability by addressing serviceability and thermal stress.
- The article explains that hardware-dense boards can create hot spots and that moving lasers to a more favorable environment can improve lifetime compared with internal laser placement.
- It also notes eye-safety and power-delivery limits for external laser source architectures.
