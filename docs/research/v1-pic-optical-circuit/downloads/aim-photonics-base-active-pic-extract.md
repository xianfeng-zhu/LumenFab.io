# AIM Photonics Base Active PIC — Extracted Details

**Source URL:** https://www.aimphotonics.com/base-active-pic
**Date accessed:** 2026-05-24

## Layer Stack

- **Three waveguide layers**: 1 silicon + 2 silicon nitride
- **Germanium photodetector** included in the standard stack
- **Metallization**: 2 copper wiring levels + 1 aluminum termination pad level
- **6 implant levels** for active component design (modulators, phase shifters, switches)

## Performance Targets

- Si waveguide propagation loss: **1.95 dB/cm** target
- SiN waveguide propagation loss: **1.85 dB/cm** target

## PDK Component Libraries

- **APSUNY** (Analog Photonics): O, C, L bands
- **Spark Photonics**: 700-1300 nm
- **RFS**: basic passives
- **Dream Photonics**: multiple waveguide geometries, couplers, splitters, thermal phase shifters, attenuators, detectors

## Customization ("Bite-Size Custom")

Available modifications outside standard MPW flow:
- Adjust ridge depth
- Adjust waveguide thicknesses
- Adjust vertical spacing between waveguides
- Trench to waveguides (with/without expose)
- Extra or custom implants
- Additional wiring level

Modifications must be achievable without development work by the fab team. Integration typically adds 1-2 months.

# AIM Photonics Low-Loss Active PIC — Extracted Details

**Source URL:** https://www.aimphotonics.com/low-loss-active-pic
**Date accessed:** 2026-05-24

## Layer Stack

- **Five waveguide pathways**: 1 silicon + 2 ridge + 2 silicon nitride
- Trench with substrate release is a standard offering
- Germanium photodetector included
- 6 implant levels
- 2 copper wiring levels + Al termination with passivation oxide and pad opening
- Edge coupler (low-loss) included

## Performance

- Si waveguide loss: **0.40 dB/cm** target
- SiN waveguide loss: **0.10 dB/cm** target
- Current SiN on fully integrated build: **0.325 dB/cm @ 1550 nm**
- Future SiN: demonstrated <0.1 dB/cm @ 1550 nm on a full active run

## Fabrication Facility

- 300 mm R&D fab at SUNY Polytechnic Institute (Albany NanoTech Complex)
- "Most advanced publicly owned 300 mm wafer facility in the world"
- 125,000 sq ft of Class 1 capable cleanrooms
