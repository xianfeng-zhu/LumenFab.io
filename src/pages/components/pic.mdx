## 0. 什么是硅光 PIC

<TermNote label="光子集成电路（PIC）" note="photonic integrated circuit，在芯片或芯片级平台上集成多种光学功能，例如波导、分束器、调制器、滤波器、耦合器和探测器。" /> 是把光的传输、分配、调制、滤波和探测集成在同一片芯片上的技术。硅光 PIC 利用 <TermNote label="硅光子学（silicon photonics）" note="以硅、二氧化硅、氮化硅、锗和金属互连等半导体工艺材料实现片上光传输、调制、滤波、探测和控制的技术平台。" /> 平台，把波导、耦合器、调制器、滤波器、探测器、微加热器、金属布线和测试结构做成一套片上光路系统。

PIC 在光引擎中承担发送和接收两个方向的信号链路：

```text
发送方向（TX，电 → 光）
  EIC 输出高速电信号 → 调制器将电信号写入光载波
  激光源提供连续光 → 耦合器 → 波导 → 分束器 → 调制器 → [WDM 合波] → 输出耦合器 → 光纤

接收方向（RX，光 → 电）
  光纤 → 输入耦合器 → 波导 → [WDM 分波] → 滤波器 → 光电探测器 → TIA → EIC
```

两条链路共享同一套底层平台——材料栈、波导、耦合器、无源/有源器件、电控、封装和测试——但光信号在两条链路中的流向和处理的器件不同。

传统光学系统依靠透镜、分束器、滤波器、调制器和探测器等分立部件完成光信号处理。PIC 把这些功能压缩到微纳加工形成的芯片结构中，使光路更紧凑、更稳定，也更适合批量制造和电学测试。硅光的优势来自成熟半导体工艺：它能实现高密度波导、复杂版图、锗（Ge）探测器、金属互连和晶圆级测试（wafer-level test），也便于与电子芯片封装和电测试流程衔接。

硅（Si）是间接带隙材料，在 1310 nm / 1550 nm 通信波段难以直接形成高效率片上激光源。实际系统中的连续光由以下几类光源方案提供，按光源与 PIC 的集成方式区分：

| 光源策略 | 说明 |
|---|---|
| 外置激光器 | 通过光纤或外置激光源模块向 PIC 输入连续波光（CW light） |
| 封装内 InP 激光器 | 激光器芯片裸片或激光器阵列与 PIC 同封装或近封装耦合 |
| III-V 族异质集成 | 通过键合、倒装或其他异质集成方式把增益材料放到硅光平台附近 |
| 激光器阵列 / 光频梳光源 | 多波长系统为 WDM PIC 提供多个波长信道 |

PIC 与光源之间的接口不只是"光能不能进去"——激光器的每一个核心参数都在约束 PIC 的设计决策：

| 激光器参数 | PIC 侧受影响的决策 |
|-----------|-------------------|
| 中心波长 | 微环共振波长设计、AWG 通道中心位置、波导单模窗口 |
| 输出功率 | 决定 PIC 的功率预算与材料选择：① 功率太低——全链路损耗（耦合 + 波导 + 分光 + 调制）累加后到达探测器的光功率必须高于灵敏度门槛；② 功率太高——硅波导在 ~100 mW 以上触发 TPA，高功率段需改用 SiN 波导；③ 微环有独立上限——~0.3 mW 输入即可触发环内 TPA 谱线畸变，进环之前必须通过分光比衰减。总线到微环的耦合是按比例的（耦合系数 κ² 固定：κ² = 1% 时，输入 100 mW 则环内 1 mW，输入 5 mW 则环内 0.05 mW），因此「激光功率 × κ² ≤ 微环功率上限」是一条联合设计约束——激光功率定了，κ² 必须匹配；反过来激光功率若因温漂或老化漂移，环内功率等比例变化 |
| RIN（相对强度噪声） | 太高则接收端 SNR 不足，需更大的调制消光比或更低的 PIC 损耗来补偿 |
| 线宽 | 相干通信时决定可用调制格式阶数的上限——线宽越窄，阶数可以越高 |
| SMSR（边模抑制比） | 不足则边模功率可能泄漏进相邻 WDM 通道，直接恶化串扰 |
| 偏振 | 硅光波导默认 TE，偏振不对则耦合效率大幅下降 |
| 偏振消光比（PER） | 即使标称 TE 偏振，仍夹杂少量 TM 分量——PER 不够高时 TM 分量在耦合器上白白损失，在偏振敏感器件（光栅耦合器、微环）中产生串扰。PER 过低需考虑片上偏振分集方案 |
| 模场尺寸 / 发散角 | 决定选光栅耦合器还是边缘耦合器，以及 taper 和 SSC 的结构设计 |
| 温漂系数 | 微环和 AWG 的热调谐范围必须覆盖激光器自身的温漂 + PIC 侧的温漂 |
| 长期老化漂移 | 激光器随使用时间老化：中心波长每年可能漂移几十 pm、输出功率逐年下降。PIC 的热调谐范围和功率预算必须为设备运行 5–10 年后的退化留出裕量，而非仅覆盖出厂状态 |
| 反馈容忍度 | 光栅耦合器、端面、接头都会反射光回激光器——容忍度低则需在光路中加隔离器 |

将以上各条串起来，一个外部激光源对 PIC 的影响不是单向的"提供多少光"——它从 PIC 设计第一天起就在约束：耦合器选型（模场和偏振匹配，PER 不足时需偏振分集）→ 波导材料选择（硅还是 SiN，取决于功率是否超 TPA 上限）→ 分光网络设计（分几路、每路分多少，受微环功率上限和探测器灵敏度两头夹逼）→ 波导尺寸（决定 n_eff，进而决定微环/AWG 的目标波长能否落在激光器波长上）→ 热调谐范围（必须覆盖激光器出厂温漂 + 长期老化漂移量）→ 是否需要隔离器（取决于激光器的反馈容忍度）。PIC 设计和激光器选型是同一道方程的两边——定了一边，另一边的自由度大部分被锁死。

PIC 与相邻系统的边界也由接口参数定义：电子集成电路（EIC）提供驱动电压、跨阻放大和控制环路；光纤接口给定模场直径、偏振和耦合容差；封装结构承担光、电、热和机械对准。

## 1. 一颗硅光 PIC 里有什么

硅光 PIC 里的器件名称很多，但它们在系统中功能分工明确。下面这张表既是器件分类地图，也是后续各小节的阅读索引——每一类器件对应页面后续的一个或几个章节：

| 类别 | 常见结构 | 在光路里的作用 |
|---|---|---|
| 光输入输出接口 | 光栅耦合器（grating coupler）、边缘（端面）耦合器（edge coupler）、模场限制转换器（spot-size converter, SSC） | 把光纤或封装光路里的模式接入片上波导，也把片上光送出芯片 |
| 波导和互连 | 条形波导（strip waveguide）、脊形波导（rib waveguide）、弯曲波导、交叉波导、锥形过渡结构 | 把光从一个器件送到另一个器件 |
| 无源光器件 | 分束器、合束器、多模干涉器（MMI）、定向耦合器、微环谐振器（ring resonator）、阵列波导光栅（AWG）、滤波器 | 分束、合束、滤波、选择波长、构成干涉结构 |
| 有源光器件 | 马赫-曾德尔调制器（MZM）、微环调制器（MRM）、相移器、热调谐器、光开关、可变光衰减器（VOA） | 用电信号或温度改变相位、强度、路径或损耗 |
| 探测器 | 锗光电二极管（Ge PD）、监测光电二极管（monitor PD） | 把光变成电流，或给控制环路提供反馈 |
| 电控制结构 | 金属布线、射频电极、焊盘、微加热器驱动、监测读出 | 提供高速驱动、低速控制、测试读出和封装连接 |

以下各节按上述类别的顺序展开——从材料栈开始，沿光信号的路径穿过波导、耦合器、无源和有源器件，最后覆盖探测与电控、设计与制造、封装和测试。

## 2. 材料栈：硅、二氧化硅、氮化硅、锗和金属层

硅光 PIC 的材料栈决定器件尺寸、损耗、热敏感性、调制效率和探测能力。下面这张表概括了四种基础材料在硅光平台中的角色——理解这个分工是后续所有器件原理的前提：

| 材料层 | 典型用途 | 工程意义 |
|---|---|---|
| 硅 | 高密度波导、紧凑调制器、Ge PD 集成、微加热器周边光路 | 面积小，适合复杂集成；尺寸和温度敏感 |
| 二氧化硅 | 包层、埋氧层、隔离层 | 提供折射率差和电/热隔离 |
| 氮化硅 | 低损耗波导、滤波器、封装内或片上低损耗转接光路 | 降低长路径损耗，改善部分无源光路 |
| 金属层 | 射频电极、微加热器、焊盘、电源/地、监测读出布线 | 把光路接到 EIC、测试设备和控制系统 |

如果所有功能都用硅来做，高密度器件和低损耗布线无法兼得——硅的高折射率差让波导紧凑，但也放大了侧壁散射损耗。多层平台的分工思路正是把不同物理需求分配给最合适的材料：硅处理高密度和有源器件，SiN 处理低损耗或相位敏感路径，金属和掺杂区处理电驱动、热调谐和读出。[不同材料在带隙、折射率、损耗和工艺成熟度上的差异](../../learn/why-no-single-material-wins-everything/) 是这种分工的底层约束。

### 2.1 绝缘体上硅（SOI）：硅光最常见的基础平台

在以上材料中，SOI 是主流硅光平台的起点——imec iSiPP、AIM Photonics、AMF、ST DAPHNE、Tower PH18 等代工平台均以 <TermNote label="绝缘体上硅（SOI）" note="silicon-on-insulator。将一层单晶硅置于氧化硅绝缘层之上、下方由硅衬底支撑的三层晶圆结构。光子 SOI 与电子 SOI 的关键区别在于埋氧层厚度——前者约 2 μm 以保证光学隔离，后者仅 50–100 nm。" /> 晶圆作为起始衬底。

SOI 的本质是一块三层晶圆：顶层是约 220 nm 的单晶硅（器件层，所有光子结构在这一层定义），中间是约 2 μm 的 SiO₂ 埋氧层（BOX），底层是约 725 μm 的硅衬底（handle wafer，提供机械支撑）。在光学上，顶层硅是高折射率芯层（core），BOX 是低折射率下包层（cladding），光场被约束在硅波导附近传播。层厚、折射率差和可制造波导截面共同决定了器件的最小尺寸、弯曲半径和工艺容差，详见 [SOI 与光子 SOI](../../learn/soi-and-photonics-soi/)。

关键概念是高折射率差。硅与二氧化硅（silicon dioxide, SiO2）的折射率差异显著，光场会被强烈约束在硅波导附近——"附近"意味着光场峰值在硅芯层内，同时仍有一小部分延伸到周围的 SiO₂ 包层中，称为 <TermNote label="倏逝场（evanescent field）" note="光场从波导芯层向包层延伸、强度呈指数衰减的部分。它属于同一个光学模式在低折射率区域的尾迹。" />。高折射率差使这部分倏逝场衰减得很快，因此波导可以做得紧凑，弯曲半径也可以较小。这种强约束同时放大了工艺敏感性：倏逝场接触到刻蚀后的粗糙硅-氧化物界面时产生散射损耗，波导尺寸误差会改变有效折射率，横电偏振（TE）和横磁偏振（TM）的差异也会更明显。

### 2.2 氮化硅（SiN）波导

部分硅光平台在硅器件层上方额外集成第二层氮化硅波导（silicon nitride waveguide, SiN waveguide），与硅波导处于不同的垂直层级——两者在芯片上各自独立布线，通过垂直耦合器在需要时互连。SiN 折射率差低于硅，因此波导尺寸更大、弯曲半径更大，但传播损耗可低至 0.1 dB/cm 以下（硅波导通常 2–3 dB/cm），功率承载能力也更好——SiN 的带隙约 5 eV，在 1550 nm 波段不存在 <TermNote label="双光子吸收（TPA, two-photon absorption）" note="硅的带隙约 1.1 eV。1550 nm 的单光子能量（0.8 eV）低于带隙，不会被硅吸收；但当波导中光功率密度较高时（直波导典型阈值 ~100–300 mW，微环因共振增强效应在 ~0.3 mW 输入时即可触发环内 TPA），两个光子可以同时被吸收、产生一个电子-空穴对，这就是双光子吸收。TPA 本身造成非线性损耗，产生的自由载流子还会引起额外的自由载流子吸收（FCA）和局部发热——对微环等谐振器件来说，mW 级的腔内功率就足以导致共振波长漂移。SiN 的宽带隙从根本上不存在这一问题，因此适合承载高功率光路。" /> 问题，温度敏感性约为硅的八分之一。在实际设计中，两种波导各司其职，在同一芯片上共存：

- **选硅波导的场景**：需要紧凑布线（弯曲半径可至数微米）、高密度集成，或必须做有源器件——调制器依赖硅中掺杂形成的 PN 结，Ge 探测器依赖硅表面作为外延生长模板。硅是"做器件"的材料。
- **选 SiN 波导的场景**：需要跨越芯片的长距离低损耗传输（损耗可低一个数量级以上）、承载高光功率而不触发非线性吸收、或对温度敏感的滤波器和延迟线——SiN 的热光系数约为硅的八分之一。SiN 是"走远路"的材料。

两者通过垂直耦合器在需要时互连，硅负责做器件和紧凑互连，SiN 负责低损耗长途传输。

前两节讨论的是光在什么材料中传播——接下来讨论光在什么材料中被探测。

### 2.3 锗探测器：为什么探测常用 Ge

硅的带隙约 1.1 eV，1550 nm 的光子能量只有 0.8 eV，低于带隙——单个 1550 nm 光子不足以被硅吸收。这是硅在通信波段"透明"的物理原因，也是为什么硅做不了探测器的根本约束。

锗（Ge）解决了这个问题。Ge 的直接带隙约 0.8 eV，恰好匹配 1550 nm 的光子能量；其间接带隙更窄（0.66 eV），也能辅助吸收。因此 Ge 在通信波段有足够的吸收系数，可以高效地将光信号转化为电信号。

实际器件中，Ge 通过 <TermNote label="减压化学气相沉积（RPCVD, Reduced Pressure Chemical Vapor Deposition）" note="在低于大气压的反应腔中，将含 Ge 的前驱气体（如 GeH₄）通入并加热分解，使 Ge 原子只在暴露的硅表面沉积生长。低压环境有助于控制生长速率和选择性——Ge 只沉积在硅上，不沉积在周围的 SiO₂ 氧化层上，因此无需额外掩膜即可只在窗口内形成 Ge 岛。" /> 选择性外延生长在硅波导表面的指定窗口上，光从下方硅波导通过 <TermNote label="倏逝耦合（evanescent coupling）" note="光场并非全部锁在波导芯层内部——一小部分倏逝场延伸到周围包层中。当 Ge 层覆盖在硅波导上方时，这部分延伸到包层中的倏逝场会逐步被 Ge 吸收。光不是在一个界面上突然跳过去的，而是沿传播方向渐进地从硅波导「漏」进 Ge 层——距离越长，耦合越充分。这与对接耦合不同：倏逝耦合不需要端面对准，容差大，且沿整个探测器长度持续发生。" /> 进入 Ge 层，被吸收后产生电子-空穴对，在 p-i-n 结的偏置电场下形成光电流。这一材料分工使硅光芯片能够在片上直接集成接收端探测器以及功率监控抽头（Monitor Tap）的读出电路。

将 Ge 长到 Si 上并非易事：Ge 和 Si 的晶格常数相差约 4%，直接外延会产生大量穿透位错，成为漏电和非辐射复合的来源。量产工艺通过低温缓冲层（~350°C）+ 高温主体层（~700–750°C）+ 后退火（~800–850°C）的组合来降低位错密度，使 <TermNote label="暗电流（dark current）" note="光电探测器在没有任何入射光时自身流过的微弱电流。它来自材料缺陷产生的非辐射复合、表面漏电和热激发。暗电流不携带信号，但会贡献噪声、降低接收灵敏度和信噪比，且随温度升高而增大——在高温 CPO 环境中尤其需要控制。" /> 控制在可接受范围。即便如此，Ge-on-Si 探测器的暗电流仍比 InP 平台原生生长的 InGaAs 探测器高出 2–3 个数量级——这是材料失配的代价。选择承受这个代价，换取的是 Ge 与 CMOS 产线完全兼容、无需引入 III-V 衬底或键合工艺的能力。

**其他探测器材料。** 在 InP PIC 平台中，探测器的首选材料不是 Ge，而是与 InP 晶格匹配的 InGaAs——它吸收系数更高、暗电流更低（&lt;2 nA），长期是长距通信的首选。片上集成的纯硅探测器也有研究（利用缺陷态吸收或双光子吸收），但效率太低、带宽有限，未进入主流。GeSn 合金是近年来的新兴方向——加入 Sn 可进一步缩小带隙、扩展响应波长至 2 μm 以上，但离量产尚有距离。在硅光代工平台的主流选项中，Ge 仍是兼顾性能与 CMOS 兼容性的最优解。

Ge 在硅光材料栈中只扮演一个角色：光探测。它不做波导，也不做调制。这个分工与硅（核心器件层）和 SiN（低损耗传输层）互补——硅负责产生和操控光信号，SiN 负责低损耗运输，Ge 负责在终点将光变回电。

### 2.4 金属、掺杂区和加热器

前面三节讨论的是光在什么材料中传输和探测。但仅靠硅、SiN 和 Ge，PIC 只是一片被动光路——光可以在片上走来走去，到了终点可以被 Ge 吸收变成电信号，却无法携带任何数据离开芯片。要让它真正有用，必须实现三件事：

- **把数据写进光里。** EIC 送来的是 0 和 1 的电信号。硅不导电、也不具备线性电光效应——只有通过掺杂形成 PN 结，用电场操控耗尽区内的载流子，才能改变波导折射率、进而将电信号转为光的强度或相位变化。这就是**掺杂**的核心作用。
- **让电信号进出芯片。** 调制器需要来自 EIC 的驱动电压，Ge PD 探测到的光电流需要送回 EIC 的跨阻放大器（TIA）。金属层——从 M1/M2 的 Cu 走线到 W 接触孔到 Al 焊盘——构成了电信号进出 PIC 的唯一通道。这就是**金属层**的核心作用。
- **补偿工艺和温度带来的偏差。** 微环的共振波长可能偏了几百皮米，MZM 的两臂相位可能不对，AWG 的通带可能在高温下漂移。如果不管它们，接收端 BER 会飙升甚至链路中断。加热器提供了可调的补偿手段——哪偏了调哪里。这就是**加热器**的核心作用。

简言之：硅、SiN、Ge 让光能传能收，掺杂、金属、加热器让光能被操控。

**掺杂：形成 PN 结和低阻接触。** 把电信号变成光信号，第一步是让电信号能改变光的相位。大多数电光材料（如 <TermNote label="铌酸锂（lithium niobate, LiNbO₃）" note="一种具有强泡克耳斯效应的非中心对称晶体，广泛用于高速电光调制器。薄膜铌酸锂（TFLN / LNOI）是近年快速发展的光子集成平台，以其高带宽、低驱动电压的调制器著称。" />）靠的是 <TermNote label="泡克耳斯效应（Pockels effect）" note="线性电光效应：外加电场使晶体折射率发生与电场强度成正比的改变。这一效应只存在于非中心对称晶体中——硅的立方晶系是中心对称的，因此不具备泡克耳斯效应。" />——外加电场直接改变折射率。但硅是中心对称晶体，没有这个效应，无法像铌酸锂那样"加电压就变折射率"。硅走的是另一条路：它通过改变波导中 <TermNote label="载流子（carrier）" note="半导体中能够携带电流的粒子，主要指电子和空穴。在硅光调制器中，通过改变 PN 结耗尽区中的载流子浓度来改变硅的折射率——这称为等离子色散效应（plasma dispersion effect），是硅基高速调制的基础。" /> 的浓度来间接改变折射率（等离子色散效应）。而要精确控制载流子浓度，就必须掺杂，形成 PN 结。

具体实现上，硅器件层通过 <TermNote label="多道离子注入（ion implantation）" note="将掺杂剂离子（如硼、磷）加速后轰击进硅晶格，改变局部的导电类型和浓度。每道注入需要独立的光刻胶掩膜来定义注入区域，注入后需高温退火激活掺杂剂并修复晶格损伤。标准有源平台使用 6–8 道注入——p 型和 n 型各 3–4 个浓度级别（如 p++、p+、p 和 n++、n+、n），形成从高浓度欧姆接触到调制器 PN 结的完整掺杂剖面。" />（p 型和 n 型各 3–4 个掺杂级别）在调制器区域形成横向 PN 结——外加反向偏压改变耗尽区宽度，进而改变波导的有效折射率，实现纯电控相位调制。在电极接入区，高浓度 p++ 和 n++ 注入将金属-半导体接触电阻降到最低，减少驱动电压在界面上的浪费。注入后需要高温退火（~1000°C）激活掺杂剂——这个温度决定了它必须在 BEOL 金属互连之前完成。

**金属层：信号进出芯片的通道。** 标准硅光平台包含两层 Cu 互连（M1 和 M2）和一层 Al 焊盘，它们与下方的硅器件层之间隔着一层约 1–2 μm 的 SiO₂ 上包层。

M1 紧贴包层表面，负责加热器供电和探测器电极的就近引出。M2（~2 μm 厚）在 M1 之上，承载高速 RF 电极——行波调制器的电极在 M2 上设计为地-信号-地（GSG）共面波导，目标特性阻抗 50 Ω。最顶层的 Al 焊盘通过钝化层开窗暴露，供引线键合或倒装互连。

金属层如何穿过 SiO₂ 包层到达硅？在需要电学连接的位置——调制器的 p++/n++ 接触区、Ge PD 的电极区、加热器两端——先在包层 SiO₂ 中向下刻出深孔，孔底暴露硅或 Ge 表面，然后填充 W（钨）并 CMP 平坦化。W 柱的一端接触硅掺杂区或 Ge 电极，另一端被随后沉积的 M1 金属覆盖——这就是垂直导电通路。

**加热器：用温度控制相位。** <TermNote label="热光调谐（thermo-optic tuning）" note="利用硅的折射率随温度变化的特性（热光效应，dn/dT ≈ 1.86×10⁻⁴ K⁻¹），通过加热器局部改变波导温度来控制相位或共振波长。响应速度较慢（μs 级），通常用于补偿工艺偏差和温度漂移，而非高速数据调制。" /> 是硅光平台中与载流子调制互补的相位控制手段。一层 TiN（或 Ni）金属电阻位于波导上方约 1 μm 处，通电后热量通过 SiO₂ 包层向下传导，利用硅的热光效应改变相位。单个 π 相移功耗约 10–50 mW，响应时间 1–100 μs。加热器带来的系统成本是调谐功耗和热串扰——高密度 PIC 中几十上百个加热器同时工作，总功耗和热耦合会进入封装和运营预算。

以上电学和热学结构在芯片纵截面中的位置——掺杂在硅器件层内部，加热器在硅波导上方约 1 μm 处，M1/M2 在加热器之上——将在下一节按从下到上的层叠顺序展开。

### 2.5 PIC 的物理结构：从衬底到焊盘的纵截面

将第 2.1–2.4 节讨论的材料按纵向空间关系组织起来，一颗典型的硅光 PIC 芯片可以按从下到上的层叠结构来理解。

**第 1 层：硅衬底（Si handle wafer）**——约 725 μm（300 mm 晶圆）或 675 μm（200 mm 晶圆）的单晶硅，提供机械支撑。在高速调制器下方，通常选用高阻硅（>750 Ω·cm）以减少衬底对射频信号的损耗。

**第 2 层：埋氧层（BOX）**——约 2 μm 的热氧化 SiO₂。这是 SOI 区别于普通硅片的关键层：它将上方的光波导与下方硅衬底隔开，防止光从波导芯层泄漏到衬底中。光子 SOI 的 BOX（2 μm）远厚于电子 SOI（50–100 nm），目的就是保证 1550 nm 波段的光学隔离。这一厚度在 imec iSiPP200/iSiPP300、AIM Photonics、AMF、Tower PH18 等主流代工平台中高度一致。

**第 3 层：硅器件层（220 nm 单晶硅）**——所有光子器件的"画布"。这一层通过光刻和刻蚀形成三种不同深度的结构：

- **全刻蚀（220 nm，蚀至 BOX）**：束缚最强、弯曲半径可至数微米，用于紧凑布线、微环谐振器和定向耦合器。因侧壁面积大，传播损耗相对较高（~2–3 dB/cm）。
- **深部分刻蚀（~150–160 nm，留下 ~60–70 nm 硅平板）**：束缚居中，用于多模干涉器（MMI）、光栅耦合器和部分调制器设计。
- **浅部分刻蚀（~70 nm，留下 ~150 nm 硅平板）**：束缚最弱、弯曲半径较大（数十微米），用于长距离布线和延迟线。传播损耗最低（可低于 0.3 dB/cm）。

同一层上还通过离子注入形成电学结构。标准有源平台包含 6–8 道注入（p 型和 n 型各 3–4 个掺杂级别），在调制器区域形成横向 p-n 结，在电极接入区形成高浓度 p++ / n++ 欧姆接触区。

**第 4 层：锗探测器区（选择性集成）**——在需要光电探测的位置，先在覆盖硅波导的氧化层中开出窗口、暴露硅波导顶面，然后用减压化学气相沉积（RPCVD）在窗口内选择性生长约 200–500 nm 的单晶锗。Ge 只生长在暴露的硅表面上，不生长在氧化层上，因此 Ge 块直接堆在硅波导的正上方、两者物理接触。这一紧贴关系正是倏逝耦合的前提：硅波导中的光场倏逝尾向上延伸到上方的 Ge 层中被吸收。Ge 层随后通过注入和退火形成 p-i-n 结。选择性生长的含义是 Ge 只出现在探测器区域，芯片其余位置的硅波导上方没有这层材料，后续的上包层会直接覆盖。

**第 5 层：上包层（SiO₂ ILD）**——约 1–2 μm 的 PECVD 沉积 SiO₂，覆盖整个芯片表面。它同时承担波导上包层和电学隔离层的角色：光场在垂直方向上被限制在硅芯层和上包层之间；金属电极则通过这层氧化层和下方硅波导隔开，避免金属吸收光。

**第 6 层：接触孔（W 接触孔 / via）**——通过光刻和刻蚀在 ILD 中开出通孔、直达下方的硅掺杂区或 Ge 电极表面，填充钨（W）并经 CMP 平坦化，形成从金属层到器件层的垂直导电通路。

**第 7 层：加热器金属层**——一层专用的金属层，通常为 TiN（氮化钛）或 Ni（镍），位于波导上方约 1 μm 处。TiN 电阻条通电后通过 SiO₂ 包层向下传导热量，利用硅的热光效应（dn/dT ≈ 1.86×10⁻⁴ K⁻¹）调节波导的有效折射率，进而控制相位或微环的共振波长。单个 π 相移功耗约 10–50 mW，响应时间约 1–100 μs。加热器通过 W 接触孔连接到 M1 层供电。

**第 8 层：金属互连层 M1 和 M2**——标准的 Cu <TermNote label="大马士革工艺（damascene process）" note="一种金属互连制造工艺：先在介质层中刻出沟槽图形，再沉积金属（Cu）填满沟槽，最后 CMP 磨平去除沟槽外的多余金属，只留下嵌入介质中的金属线。与之相对的传统工艺是先沉积整层金属再刻蚀——Cu 难以刻蚀，因此大马士革工艺成为 Cu 互连的标准做法。" /> 互连。M1 用于局部互连（例如加热器供电、低速控制），M2（通常 ~2 μm 厚）用于长距离布线和高速 RF 电极。调制器的行波电极在 M2 层设计为地-信号-地（GSG）共面波导结构，目标特性阻抗 50 Ω，并追求 RF 信号与光信号的群速度匹配以最大化电光带宽。

**第 9 层：钝化层与焊盘**——最顶层的 SiN 或 SiO₂ 钝化保护层。在引线键合或倒装互连区域开出 Al 焊盘窗口，在光栅耦合器区域也需开窗以供光纤从芯片上方耦合。

**可选层：SiN 波导层**（物理位置在第 3 层与第 5 层之间）——部分平台（如 imec iSiPP300 和 AIM Photonics Low-Loss）在硅器件层和上包层之间集成了 PECVD 或 LPCVD SiN 波导（约 0.4 μm 厚），作为第二层低损耗波导材料，与硅波导共同构成多层光路。

将以上层次汇总，一颗完整有源硅光 PIC 芯片的工艺制造需要约 15–20 道光罩（mask levels）——其中硅图形化 3 道、离子注入 6–8 道、Ge 模块 2–3 道、接触孔 1 道、加热器金属 1 道、M1/M2 互连 2 道、钝化焊盘 1 道、深沟槽 1 道。仅含无源器件的简化平台约需 8–10 道光罩。这些光罩的具体内容和顺序在第 8.4 节的工艺流程中展开。

## 3. 波导与模式：片上光怎样被约束

波导是芯片上的导光结构。光场依靠芯层与包层之间的折射率差被约束，并以 <TermNote label="光学模式（optical mode）" note="满足波导或光纤边界条件的稳定光场分布。模式决定光场大小、形状、偏振和传播常数。" /> 的形式沿特定几何结构传播。有效折射率、模场尺寸和偏振是贯穿波导设计、耦合器选择和器件仿真的三条主线，详见 [波导与光学模式](../../learn/waveguides-and-optical-modes/)。

### 3.1 波导怎样限制光

波导限制光的物理基础是全内反射：光从高折射率介质（芯层）进入低折射率介质（包层）时，若入射角大于临界角，光无法穿出界面，被全部反射回芯层。硅波导正是利用硅（n ~3.5）与 SiO₂（n ~1.45）之间悬殊的折射率差，将光场强烈约束在硅芯层附近——光在垂直和横向两个方向上被"困住"，只能沿波导长度方向传播。

如 §2.1 所述，光场并非 100% 锁在硅内部——一小部分倏逝场延伸到周围的 SiO₂ 包层中。硅与 SiO₂ 之间的折射率差越大，倏逝尾在包层中衰减越快，光场越集中在硅芯层附近——这是硅波导能做得如此紧凑的根本原因。

光场同时跨硅和 SiO₂ 两种材料，整个模式看到的等效折射率称为有效折射率（effective index）——它是硅和 SiO₂ 折射率的加权平均，落在 1.45–3.5 之间。n_eff 决定光在波导中的传播速度（v = c / n_eff），也决定微环的共振波长、MZM 的相位积累和 AWG 的波长路由——它是连接波导物理结构与器件性能的核心参数。

### 3.2 波导宽度、高度和偏振

波导的截面尺寸与波长之比决定了它支持几个空间模式。标准的 220 nm 高硅波导，宽度在约 400–500 nm 时满足 <TermNote label="单模条件（single-mode condition）" note="波导只允许一个空间模式传播的状态。对于 220 nm 高的硅波导，宽度小于约 500 nm 时满足单模条件。单模波导的相位行为确定、可预测，是大多数硅光器件的设计前提；超出单模条件后，多个模式以不同速度传播、互相干涉，可用来构建 MMI 等器件。" />——给定波长下只支持一个 <TermNote label="TE 模式（transverse electric）" note="横电模式——电场主要平行于芯片表面（水平方向），磁场有纵向分量。硅波导中 TE 模式束缚更强、损耗更低，是大多数硅光器件默认的工作偏振。" /> 和一个 <TermNote label="TM 模式（transverse magnetic）" note="横磁模式——磁场主要平行于芯片表面，电场有纵向分量。由于硅波导截面不对称（高 220 nm ≠ 宽 400–500 nm），TM 的有效折射率和损耗与 TE 不同。" />。单模波导中只有一个模式能传播，光走多远、积累多少相位是确定的——这是 MZM、微环、AWG 等相位敏感器件能稳定工作的前提。多模波导则允许多个模式同时存在，每个模式的有效折射率不同，走到出口时相位各不一样，叠加后形成干涉图案——MMI 正是利用这种多模干涉来精确分光或合光。但这也意味着如果波导尺寸意外激发了不需要的模式，它的干涉就会变成串扰。

在单模波导中，TE 和 TM 两种偏振的行为并不相同。原因在于硅波导的截面不是正方形——宽约 400–500 nm、高 220 nm——水平和垂直方向的几何边界不同，两种偏振「看到」的波导形状不一样，有效折射率和侧壁散射自然有差异。因此许多硅光器件会指定工作偏振（通常优先选 TE，因其在硅波导中束缚更强），尤其是光栅耦合器、微环谐振器、AWG 和紧凑分束器。封装和光纤耦合也要把偏振方向纳入接口条件。

### 3.3 损耗、弯曲和串扰

波导不是理想的无损通道——光在里面传播时面临四类工程约束，这四类约束分别限制了版图的布线密度、最小弯曲半径和波导间距：

| 工程问题 | 来源 | 对 PIC 的影响 |
|---|---|---|
| 传输损耗（propagation loss） | 侧壁粗糙、材料吸收、工艺缺陷、包层污染 | 光功率随路径变长下降，压缩链路预算 |
| 弯曲损耗（bend loss） | 转弯半径过小，模式向外泄漏 | 影响版图密度和布线自由度 |
| 信道间串扰（crosstalk） | 相邻波导距离太近，倏逝场（evanescent field）互相耦合 | 造成相邻信道串光、滤波泄漏或光开关隔离度下降 |
| 偏振敏感性 | 波导截面不对称造成 TE / TM 差异（见 §3.2），应力进一步放大 | 影响耦合、滤波和封装偏振要求 |

### 3.4 工艺误差如何改变光路

硅光波导对尺寸误差极为敏感——高折射率差让波导紧凑，但也意味着同样的宽度偏差造成的有效折射率变化远大于 SiN 或光纤等低折射率差平台。

光凭数字来看：标准的 220 nm 高、400–500 nm 宽的条形波导，1 nm 的宽度偏差可导致有效折射率变化约 3×10⁻³，折算到微环共振波长上约 1 nm 的漂移。对于通道间隔仅 0.8 nm（100 GHz）的密集 WDM，一个通道的信号会直接落入相邻通道。加宽波导到 2 μm 可将敏感度压低 100 倍，代价是弯曲半径变大、集成密度下降——这是工艺容差与集成密度之间的基本取舍。

实际代工中，标准 MPW 流片的波导宽度平均偏大约 27 nm，属于光刻和刻蚀的系统偏移而非随机噪声——版图上的宽度不等于芯片上的宽度。同一晶圆中心和边缘之间、不同批次之间还存在随机波动。这些偏差进入有效折射率后，微环、MZM、AWG 等相位敏感器件的性能直接偏离设计值。

补偿手段在 §2.4 中已有铺垫——热调谐器修正相位、监测 PD 追踪漂移、晶圆测试数据反馈给 PDK——三者构成「设计-制造-校准」闭环。完整版工艺误差表见 §8.5。

汇总以上，波导的几何尺寸和光学参数是整个硅光平台的底层变量——几乎所有后续器件的性能都直接或间接地建立在这些波导参数之上：

| 波导参数 | 物理影响 | 后续依赖它的器件与指标 |
|----------|----------|------------------------|
| 宽度、高度 | 决定有效折射率 n_eff | MZM 的相位积累、微环的共振波长、AWG 的通道位置 |
| 蚀刻深度（全/深/浅） | 决定光场束缚强度和弯曲半径 | 集成密度、波导交叉损耗、MMI 多模干涉条件 |
| 侧壁粗糙度 | 决定传播损耗（dB/cm） | 链路功率预算→激光器功率需求 |
| 截面不对称（高≠宽） | 造成 TE/TM 的有效折射率和损耗差异 | 偏振敏感器件（光栅耦合器、微环、AWG）的设计约束 |
| 宽度偏离设计值 | 转化为 n_eff 误差→相位误差 | MZM 偏置点漂移、微环共振波长偏移、AWG 通道失准 |

## 4. 光输入输出：从光纤到片上波导

光纤与片上波导的模场尺寸相差悬殊——硅波导的模式被强烈约束在亚微米级的硅芯层内（硅与 SiO₂ 之间的折射率差约 2.0），而单模光纤的 <TermNote label="模场直径（MFD）" note="mode-field diameter。它描述单模光纤中主模式光场的有效直径，比几何芯径更能反映耦合时看到的光场大小。" /> 约 9–10 μm（光纤的芯包折射率差仅 ~0.005）。这相当于把一根水管接到一根毛细血管上——两者之间需要耦合器、锥形过渡结构、SSC 或透镜来渐进过渡。硅光平台提供两种主流耦合方案：光栅耦合器（从芯片表面耦合）和边缘耦合器（从芯片端面耦合），它们的取舍贯穿设计、测试和封装全过程。

```text
大尺寸光纤模式
→ 锥形结构 / 光栅结构 / 透镜过渡
→ 小尺寸硅波导模式
```

### 4.1 光栅耦合器：从芯片表面耦合

光栅耦合器是在硅波导自身上刻出周期性凹槽（利用硅器件层的浅刻蚀 ~70 nm，在 220 nm 厚的硅上挖出周期 600–1800 nm 的光栅），光沿波导传播时每遇一个凹槽就有一小部分被向上散射，所有散射在特定角度上相位叠加，形成指向芯片表面外的衍射光束，将片上导模向上衍射到芯片表面外，光纤从上方以 8–15° 的倾角对准——同一结构在相反方向上也能将光纤的光耦合进波导。

其核心优势是晶圆级测试便利——光纤从芯片上方即可对准，无需切割和端面加工，适合 MPW 流片的早期筛选。代价是性能：标准光栅耦合器的插损约 3–5 dB，1 dB 光学带宽约 30–40 nm，对偏振敏感（TM 效率远低于 TE），且标准结构因布拉格对称性存在 >5% 的后向反射——返回激光器的光可能扰动其单模稳定性。高级设计（切趾光栅、闪耀光栅、双层多晶硅覆盖光栅）可将插损压到 &lt;1 dB、带宽扩展到 >100 nm、后向反射压到 &lt;1%，但增加了工艺复杂度。同时，晶圆级测试中引入的额外光栅也会消耗信号功率（通常 1–5%），是不可逆的链路功率代价。

### 4.2 边缘（端面）耦合器：从芯片边缘耦合

边缘（端面）耦合器沿芯片平面方向工作，核心结构是反向锥形（inverse taper）：硅波导的尖端被缩窄到 &lt;180 nm（接近光刻最小线宽），光场因波导太窄而无法被有效束缚，从硅芯层向外膨胀进入周围的 SiO₂ 或 SiN 包层中；随后一个 SiN 波导或聚合物辅助层接住这个膨胀后的模式，进一步将其扩束到与单模光纤的 ~9–10 μm 模场匹配。整个过渡沿 200–600 μm 的长度渐进完成（绝热过渡），将光送入或送出 PIC。

与光栅耦合器相比，边缘耦合器通常有更宽的光学带宽（>100 nm 1dB）、更低的耦合损耗（典型 0.5–1.5 dB/端面，最优 &lt;0.2 dB），且偏振相关损耗更小。代价是它需要高质量芯片端面（通过深沟槽刻蚀或解理暴露）和精密对准——直接耦合的 1dB 横向对准容差仅 ±1–2.5 μm，通常需要光纤阵列单元（FAU）、光学胶和主动对准设备配合完成封装。

### 4.3 耦合损耗为什么重要

耦合损耗（coupling loss）同时出现在入射和出射两条路径上，每 1 dB 耦合损耗等比例压缩 1 dB 的接收端光调制幅度（OMA）。举例来说，若每个端面的耦合损耗从 1.5 dB 升到 3 dB，两条路径合计多损失 3 dB——相当于接收端可用的光功率直接砍半，系统要么提升激光器功率（增加热负载），要么承受 BER 恶化。

在共封装光学（CPO）中，高通道数放大了耦合损耗的影响：损耗的平均值决定链路预算能不能闭合，损耗的长尾分布决定封装良率——几百个通道中只要少数几个超规格，整颗芯片可能报废。因此边缘耦合器在 CPO 中更受青睐（损耗更低、带宽更宽），而光栅耦合器主要在晶圆级测试阶段发挥作用。

**光栅耦合 vs 边缘耦合：行业现状。** 当前硅光产品的标准做法是两者分工——晶圆级测试用光栅耦合器从上方快速筛选，最终组装用边缘耦合器满足链路预算。对于 OMA 预算仅约 5–6 dB 的典型 CPO 链路，光栅耦合器在最终组装中不具备可行性（发送+接收合计 6–10 dB 就已超预算）。耦合适配方案的选择不是 PIC 设计者或封装方某一方单独决定的：PIC 设计者在版图上选定耦合器类型，封装方的对准能力和设备决定了制造成本和良率，系统方的链路预算决定了光栅耦合器在最终组装中根本不在选项之内。近年趋势是混合 GC+EC 结构（同一位置兼具测试与组装功能）和 SiN 边缘耦合器（sub-1 dB 损耗成为 CPO 标准，且与中介层上的聚合物光再分布兼容）。

从更大视角看，耦合是光模块全链路中几乎无法事后补救的一环。调制器偏了可以调偏置、微环偏了可以热调谐、AWG 偏了可以温控——耦合偏了，光直接就丢了，没有任何片上补偿手段。它也是产业链的物理交汇点：PIC 制造是硅工艺、精度在纳米级；光纤对准是机械工艺、精度在微米级——两者之间没有缓冲区。芯片设计得再好，耦合过不去，整个模块等于零。

## 5. 无源光路器件：分束、合束、滤波和路由

<TermNote label="无源光器件（passive optical devices）" note="passive optical devices。不需要外部电驱动即可工作的光学元件——通过材料的几何形状和折射率分布对光进行分束、合束、滤波、路由等操作，不向光信号注入能量。" /> 通过几何结构和折射率分布改变光的路径、相位关系、功率分配和波长选择。它们是硅光 PIC 的基础组件库——调制器、光开关、WDM 系统乃至监测抽头都由这些基础单元搭建而成。

无源器件按功能可分为两类：**功率分配器件**（Y 分支、定向耦合器、MMI）——将光功率按比例分到多路或合路，分光比由几何结构决定，与波长无关；**波长选择器件**（微环谐振器、AWG、阶梯光栅）——利用干涉或谐振将不同波长导向不同的输出端口，是 WDM 系统的核心。

这两类器件在设计时都与激光源参数直接关联：功率分配器决定每路光功率，分几路、每路分多少直接取决于激光器输出功率能支撑多少支路同时满足探测器的灵敏度门槛（同时还要守在硅波导的 TPA 上限之下）；波长选择器则要求激光器的中心波长、线宽、SMSR 和温漂落在器件可接受的设计窗口内（回顾 §0 的激光-PIC 接口表）。无源器件的设计无法独立于激光源——分多少路取决于激光器能提供多少功率，微环和 AWG 的目标波长必须对准激光器的中心波长。反过来，激光器一旦选定，分光网络的结构和波长选择器件的设计窗口也就随之确定。

### 5.1 分束器 / 合束器：分光和合光

分束器（splitter）把一路光分成多路，合束器（combiner）反之。硅光平台中三种主流结构各有其物理基础。

**Y 分支（Y-branch）** 是最直接的分光方式：一根波导逐渐加宽，然后沿中线分叉为两根镜像对称的输出波导。光在波导中沿着物理边界走——如果分叉角度足够小（通常 0.5–1°），光场有足够的时间和空间从一个宽模式平滑地"滑"进两个分支各自的模式中，过程中几乎没有能量泄漏。这个过程称为绝热过渡。如果分叉太陡，光来不及跟随波导形状变化，一部分功率会逸出成为辐射损耗。优化良好的 Y 分支附加损耗仅 0.1–0.5 dB，分光比由几何对称性自然保证，不依赖波长。

**定向耦合器（§5.2）和 MMI（§5.3）** 则通过不同的干涉机制实现分光，各自在分光比精度、带宽和工艺容差上有所取舍。

这些基础分光/合光单元不仅独立使用，还会作为子结构嵌入 MZM（两臂前的分光和合光）、光开关网络和监测抽头中。分光比、附加损耗、带宽、相位平衡和偏振相关性决定了这些基础单元能否在复杂光路中保持一致性。

### 5.2 定向耦合器：靠相邻波导交换能量

"定向"的含义是功率只从一条波导向另一条波导的**同向**端口转移——光从波导 A 的输入侧进入，只会耦合到波导 B 的前向端口（交叉端），不会反向耦合回波导 A 的输入侧。功率流动有确定方向，这是与普通分束器的本质区别。

当两条相同的波导平行排列、间距缩小到几十到几百纳米时，一条波导的倏逝场进入另一条波导的芯层，光功率在两条波导之间周期性转移——波导 A 中的光逐渐衰减、波导 B 中的光同步增长，走过一段距离后又倒回来。功率完全从一条波导转移到另一条所需的距离称为耦合长度 L_c，典型的 SOI 定向耦合器（宽 500 nm、高 220 nm、间距 ~200 nm）L_c 约 13 μm。功率随传播距离正弦振荡：耦合区长度取 L_c 时全部转移至交叉端，取 L_c/2 时实现 50:50 的 3-dB 分光，超过 L_c 后功率再次耦合回原波导。

定向耦合器的分光比对两个变量极其敏感。一是波导间距——偏差几纳米即可显著改变 L_c，版图中通常预留测试结构和设计裕量。二是波长——倏逝场的延伸范围随波长增大而扩大，在 1.5–1.6 μm 范围内 L_c 可变化约 58%。设计在 1550 nm 为 50:50 的定向耦合器，到了 1530 nm 分光比可能偏离到 45:55。这意味着激光器中心波长一旦确定，定向耦合器的设计窗口也就随之锁死（回顾 §0 的激光-PIC 接口表）。正是这种波长敏感性，使定向耦合器在需要宽带平坦响应的场景下被 MMI 取代（§5.3）。

交叉端输出始终比直通端滞后 90° 相位——因为交叉端的光来自倏逝耦合，经历了一个等效的 90° 延迟。这个固定的 π/2 相移使定向耦合器在 MZM（两臂需要精确的相位关系）和微环总线耦合区（环与总线的耦合）中不可替代。

### 5.3 MMI（多模干涉器）

MMI 全称 Multimode Interference，即多模干涉器。它的结构很简单：一段宽波导（多模区），宽度通常几微米、长度几十微米，两端各接一组单模波导作为输入和输出。光进入多模区后，同时激发多个横向模式，这些模式以略微不同的速度向前传播，在特定距离上重新叠加时，相位关系恰好让输出端形成若干个功率相等的像——这就是天然的 1×2、2×2 或 1×N 分光/合光器。

MMI 的物理基础是自成像效应（self-imaging）——导波光学中的 Talbot 效应。多模区中各模式的传播常数近似按二次序列排列，使得光场在传播一定距离后自动复现输入图案。这个特征距离由拍长 L_π 决定：

```
L_π ≈ 4 n_c W_e² / (3 λ₀)
```

自成像位置只取决于多模区长度与 L_π 的比例——设计 1×2 时取长 3L_π/2，两个输出端在 ±W_e/2 处；设计 2×2 时两输入端偏移 ±W_e/6，输出携带 90° 相位差；设计 1×N 时长度缩放为 3L_π/N。

> **L_π 的参数含义与典型数值：** n_c 为多模区折射率，W_e 为等效宽度（比几何宽度略宽，因为光场会少许渗入包层），λ₀ 为自由空间波长。以典型的 SOI 平台为例，多模区宽 3 μm、n_c ≈ 2.8，得 L_π 约 25 μm，1×2 MMI 长度约 3L_π/2 ≈ 38 μm。

MMI 的核心优势在工艺容差。分光比由多模区的长度和宽度这两个几何参数决定——不像定向耦合器那样依赖两条波导之间的纳米级间距。加之自成像条件具有周期性（小幅长度偏差只轻微偏移最优工作点），MMI 在工艺波动下比定向耦合器稳定得多。典型的超额损耗约 0.1–0.5 dB，分光比精度优于 ±0.5 dB，光学带宽可达 100 nm 以上。代价是面积——1×2 MMI 约三十到四十微米长，而同等功能的定向耦合器只需十几微米。

### 5.4 微环谐振器：用微环选择波长

微环是一个封闭的波导环路，通过总线波导与外界交换光。光在环内循环——只有波长满足"环程光程 = 波长整数倍"这一条件的模式，才能每走一圈叠加一次，在环内相长干涉形成谐振：

```
m · λ_res = n_eff · 2πR
```

其中 m 为整数（谐振阶数），λ_res 为共振波长，n_eff 为波导有效折射率，R 为环半径。不满足此条件的波长通过总线波导后几乎不衰减。

谐振峰在光谱上周期排列——相邻两个共振峰之间的波长间隔称为自由光谱范围（FSR）：

```
FSR_λ = λ² / (n_g · L)
```

其中 n_g 为群折射率，L = 2πR 为环周长。核心取舍：环越小 → FSR 越大 → 单个 FSR 内能容纳更多 WDM 通道；但环越小 → 弯曲损耗越大 → Q 值越低。

品质因数 Q = λ / Δλ 描述谐振峰的尖锐程度——Δλ 越小，Q 越高，波长选择性越好，但环内光子寿命越长，调制速度越慢。Q 值与环的总损耗（弯曲辐射、侧壁散射、材料吸收）及总线耦合强度共同决定。

**临界耦合（critical coupling）** 是微环设计的核心概念。当总线到环的耦合强度恰好等于环内往返一圈的总损耗时，谐振波长上的光被完全耦合进环内并耗散掉，直通端输出为零——消光比最大。耦合过弱（欠耦合），光进环太少；耦合过强（过耦合），谐振峰变宽、消光比下降。

微环可以做成两种构型：仅有直通端的 all-pass 构型用于强度调制和传感；带有下载端（Drop Port）的 add-drop 构型则可将谐振波长从总线中"下载"到另一条波导上，用于 WDM 信道分出/插入。

微环占用面积极小（半径可至几微米），是硅光平台上最紧凑的波长选择器件——但代价是对温度极其敏感（§2.4 和 §3.4 已讨论补偿手段）。

| 微环指标 | 含义 |
|---|---|
| 自由光谱范围（FSR） | 相邻共振峰之间的波长间隔，决定一个 FSR 内可排列多少 WDM 通道 |
| 品质因数（Q factor） | 谐振峰的尖锐程度，决定波长选择性 |
| 插入损耗（insertion loss） | 非谐振波长经过器件后的残余损耗 |
| 消光比（extinction ratio） | 谐振开/关状态下的光强比，由耦合强度与环内损耗的相对关系决定 |
| 热调谐范围 | 微加热器能够拉动共振波长的范围 |

### 5.5 AWG / 阶梯光栅：多波长复用和解复用

波分复用（WDM）系统中，不同波长可以在同一根光纤或同一条波导里传输——但最终必须在发射端合到一起、在接收端重新分开。AWG（阵列波导光栅）和阶梯光栅是硅光平台上完成这一功能的两类核心器件。

**AWG 的结构与工作原理。** AWG 由三部分组成：输入星形耦合器（自由传播区 slab）、波导阵列、输出星形耦合器。光从输入波导入射到第一个 slab 区后自由衍射，均匀分配到阵列中的每根波导。相邻阵列波导之间存在恒定的长度差 ΔL，使得相邻通道在到达输出 slab 时携带一个与波长相关的固定相位差。这一线性相位梯度在输出 slab 中形成倾斜波前——不同波长的波前倾斜角不同，因而聚焦到不同的输出波导位置。

中心波长满足光栅方程：

```
n_c · ΔL = m · λ₀
```

其中 n_c 为阵列波导有效折射率，m 为衍射级次。ΔL 是 AWG 最核心的设计参数——它决定了哪个波长聚焦到中心输出口。

通道间隔由输出波导间距、slab 焦距和阵列波导间距共同决定。AWG 的自由光谱范围 FSR = λ₀·n_c/(m·n_g)，必须大于通道数 × 通道间隔，否则不同衍射级次的波长会在输出端重叠。

AWG 的主要工程挑战是相位误差——阵列波导之间的随机相位偏差（来自波导宽度和厚度的工艺波动）是串扰的主要来源，典型串扰水平在 −20 至 −30 dB。温漂是另一项关键约束，通常需要热调谐或非热化设计来稳定通道位置。

**阶梯光栅（Echelle grating）** 用单个自由传播区和一片反射式光栅实现同样的功能——光从输入波导发散、被光栅齿面反射并色散、不同波长聚焦到不同输出位置。相比 AWG，阶梯光栅的面积更紧凑，但依赖高反射率光栅齿面（由金属镀膜或利用 Si-SiO₂ 界面的全内反射实现），插损和串扰对光栅刻蚀精度更敏感。

## 6. 有源光器件：用电信号控制光

<TermNote label="有源光器件（active optical devices）" note="active optical devices。需要外部电驱动或热控制来工作的光学元件——通过注入电流、施加电压或加热来改变波导的折射率，从而控制光的相位、强度、路径或波长。" /> 把电信号、热信号或控制状态转换为光学变化。调制器、相移器、热调谐器、光开关和 VOA 都属于这一层。电光转换的核心约束——调制电压、带宽和能耗之间的权衡——决定了调制器结构的选择，详见 [电信号到光信号](../../learn/electrical-signal-to-optical-signal/)。

### 6.1 相位调制：改变折射率就是改变相位

光在波导中传播时会积累相位。改变有效折射率，就会改变相位。硅光里常见的相位控制方式包括载流子调制和热光调谐；混合集成平台还可能引入更强的电光材料。相位变化可以通过干涉转换为强度变化，也可以用来调滤波器、光开关矩阵或光束扫描。

### 6.2 马赫-曾德尔调制器，MZM

MZM 把一路光分成两臂，两臂积累不同相位，再合束。相位差改变后，相长或相消干涉会改变输出光强。EIC 驱动器送来的高速电信号调制其中一臂或两臂的相位，从而形成高速强度调制。

| MZM 指标 | 工程含义 |
|---|---|
| VπL | 达到 π 相移所需电压与长度的乘积 |
| 插入损耗 | 调制器本身消耗多少光功率 |
| 消光比 | 高低光电平之间的差 |
| 3 dB 电光带宽（3 dB Electro-optic Bandwidth） | 电极和相位调制能支持的频率范围 |
| 啁啾（chirp） | 调制时相位与强度耦合程度 |
| 每比特能耗 | 每 bit 调制消耗多少电能 |

### 6.3 微环调制器

微环调制器利用微环共振位置的移动来改变输出光强。它面积小、驱动电容小、能耗潜力好，也天然适合高密度 WDM。工程代价是工作点窄：温度变化和工艺漂移都会移动共振波长，因此热调谐、监测和闭环控制常常成为器件设计的一部分。光学带宽和消光比也要与目标调制格式共同设计。

### 6.4 热调谐器和相移器

硅的折射率会随温度变化，微加热器可以调节相位或共振波长。微加热器常用于校准工艺误差、锁定微环、调节 MZM 偏置、修正 AWG 通带或控制光开关矩阵。它带来两项系统成本：调谐功耗和热串扰。高信道数 PIC 中，微加热器的总功耗和热串扰会进入封装与运营预算。

### 6.5 光开关和 VOA

光开关改变光的路径，可变光衰减器（VOA）调节某一路光的功率。它们可以由 MZM、微环、热相移器或载流子相移器构成。可重构光网络、片上互连、测试矩阵、相控阵和功率均衡结构都会用到这类器件。

## 7. 从光到电，再从电控制光

硅光 PIC 是光学结构与电子控制共同工作的芯片。它把光路、光电探测器、监测结构、微加热器、驱动器、跨阻放大器（TIA）和控制环路放在同一个系统里。

### 7.1 锗光电二极管：把光变成电流

如 2.3 节所述，硅在通信波段吸收弱，PIC 选用锗材料实现片上探测。Ge PD 吸收通信波段的光子，产生电子-空穴对，并在电场下形成光电流。后级 TIA 把这个电流转换成电压。响应度、暗电流、带宽、饱和光功率、电容、噪声和线性范围决定探测器能否把光信号稳定交给接收电路。

### 7.2 监测光电二极管：给控制系统提供反馈

复杂 PIC 通常需要内部监测：监控输入光功率，辅助微环锁定，检查耦合器状态，校准 MZM 偏置，平衡 WDM 信道。监测光电二极管的读数进入固件和控制环路，用来把温漂、工艺漂移和封装漂移变成可校准的控制量。

### 7.3 驱动器、TIA 和控制电路

调制器需要驱动器，光电二极管需要 TIA，微加热器需要数模转换器（DAC）或电流驱动器，监测环路需要模数转换器（ADC）和控制算法，高速接口需要射频设计。PIC 与 EIC 的距离、焊盘映射、微凸点、地回路（Ground Return）完整性、ESD 和测试通路，会决定实验室里的器件指标能否在封装后保持。

### 8.4 工艺流程

完整的硅光 PIC 制造流程可以按前道工艺（FEOL）和后道工艺（BEOL）两阶段来理解。以下以典型的 imec iSiPP50G / AIM Photonics Base Active 有源平台为参照，这类平台支持三种蚀刻深度的硅波导、离子注入调制器、Ge 光电探测器和多层金属互连，总计约 15–20 道光罩。仅含无源器件的简化平台可缩减至 8–10 道光罩。

**前道工艺（FEOL）——从 SOI 晶圆到光探测器**

**（1）SOI 晶圆准备。** 200 mm 或 300 mm SOI 晶圆，由 Smart-Cut（Unibond）工艺制备：220 nm 顶层单晶硅 / 2 μm 热氧化 BOX / ~725 μm 硅衬底。光子 SOI 的 BOX 远厚于电子 SOI（后者仅 50–100 nm），以保证 1550 nm 波段的光学隔离。衬底可选用高阻硅（>750 Ω·cm）以降低 RF 损耗。

**（2）硅波导图形化——3 道蚀刻，3 道光罩。** 这是硅光工艺区别于 CMOS 的最核心步骤之一。在 220 nm 的顶层硅上，通过三次独立的光刻和反应离子刻蚀（RIE），形成三种不同深度的波导结构：

| 蚀刻类型 | 典型深度 | 剩余硅平板 | 形成的结构 |
|----------|----------|-----------|-----------|
| 浅部分蚀刻 | ~70 nm | ~150 nm | 低损耗脊形波导、光栅耦合器 |
| 深部分蚀刻 | ~150–160 nm | ~60–70 nm | 标准脊形波导、MMI、锥形过渡 |
| 全蚀刻 | 220 nm | 0（至 BOX） | 条形波导、微环、定向耦合器 |

光刻采用 193 nm 浸没式 DUV，最小特征尺寸可至 ~90 nm。刻蚀要求侧壁角 >88°、侧壁表面粗糙度 &lt;2 nm。刻蚀后通常辅以化学处理或氢退火平滑侧壁，降低散射损耗。三次蚀刻的套刻精度（overlay）直接影响器件性能——波导宽度和位置的偏差会转化为有效折射率的变化。

**（3）SiN 波导沉积与图形化（可选，0–2 道光罩）。** 部分平台在硅波导完成后，通过 PECVD 或 LPCVD 沉积约 0.4 μm 的 SiN，并图形化为第二层低损耗波导。PECVD 工艺温度较低（~400°C），与已形成的掺杂结兼容性更好；LPCVD 可达到更低的传播损耗（&lt;0.1 dB/cm）和更好的厚度均匀性。

**（4）离子注入——6~8 道光罩。** 这是形成调制器和低阻接触区的关键步骤。通过多道光刻胶掩膜，对硅器件层的特定区域注入 p 型和 n 型掺杂剂：

- 调制器区：中等浓度的 p 型和 n 型注入，在波导中形成横向 p-n 结。调制器工作在载流子耗尽模式——外加反向偏压改变耗尽区宽度，从而改变波导的有效折射率。
- 接触区：高浓度 p++ 和 n++ 注入，降低后续金属-半导体接触的电阻。

标准有源平台（AIM Photonics、AMF）使用 6 道注入：p++ / p+ / p 和 n++ / n+ / n。imec 平台使用 8 道注入（p 型和 n 型各 4 个浓度等级），以实现更精细的掺杂剖面控制。注入后需在 ~1000°C 下退火激活掺杂剂并修复晶格损伤。退火必须在后续金属层工艺之前完成——BEOL 的 Cu 互连无法承受此温度。

**（5）Ge 光电探测器选择性集成——2~3 道光罩。** 这是 PIC 中工艺最复杂的单一模块：

- 先在波导上方的氧化层中光刻和刻蚀出窗口，暴露硅波导表面。
- 用 RPCVD 在窗口内选择性外延生长单晶 Ge。Ge 只生长在暴露的硅上，不生长在氧化层上。典型工艺：低温缓冲层（~350°C）→ 高温主体层（~700–750°C）→ 后退火（~800–850°C）以降低穿透位错密度。
- CMP 平坦化，最终 Ge 厚度约 200–500 nm。
- 随后进行 n+ 和 p+ 注入，在 Ge 层中形成横向或垂直 p-i-n 结，并退火激活（&lt;700°C）。

光从下方硅波导通过倏逝耦合进入 Ge 层被吸收，无需对接耦合或额外对准结构。

**后道工艺（BEOL）——从接触到焊盘**

**（6）上包层沉积与接触孔形成——1 道光罩。** 沉积约 1–2 μm 的 PECVD SiO₂ 作为上包层和电学隔离层。在需要电学连接的位置——调制器的 p++/n++ 区、Ge PD 的电极区、加热器两端——光刻和刻蚀开出通孔，填充 W（钨）插塞并 CMP 平坦化。该步骤同时形成硅化物（如 NiSi），降低金属-半导体接触电阻。

**（7）加热器金属层——1 道光罩。** 沉积并图形化 TiN（或 Ni、W），形成微加热器电阻条。加热器位于波导上方约 1 μm 处，由 SiO₂ 上包层隔开以避免金属吸收光。热光相移器（thermo-optic phase shifter）利用硅的热光效应（dn/dT ≈ 1.86×10⁻⁴ K⁻¹）调节相位。

**（8）金属互连 M1 和 M2——2 道光罩。** 标准 Cu 大马士革工艺形成两层金属互连。M1（~200 nm–1 μm）用于就近引出和加热器供电；M2（~2 μm Al 或 Cu）用于长距离布线和高速 RF 电极。调制器区域的 M2 层形成地-信号-地（GSG）共面波导行波电极，目标特性阻抗 50 Ω，RF 和光信号的群速度匹配决定电光带宽的上限。部分高级平台（imec iSiPP300、ST DAPHNE）提供多达 4–6 层金属。

**（9）钝化与焊盘开窗——1 道光罩。** 沉积 SiN / SiO₂ 钝化层保护芯片表面。在引线键合或倒装互连对应位置光刻和刻蚀开出 Al 焊盘窗口；在光栅耦合器区域也需开窗，供光纤从芯片上方对准耦合。

**（10）深沟槽刻蚀（边缘耦合平台）——1 道光罩。** 通过深反应离子刻蚀（DRIE）将芯片边缘的 SiO₂ 和部分衬底去除，暴露波导端面，形成平整的光学端面供光纤端面对准耦合。仅使用光栅耦合器作为光 I/O 的平台可跳过此步骤。

**（11）晶圆级测试、划片与最终测试。** 通过片上光栅耦合器和探针焊盘，对每颗芯片进行晶圆级光学和电学测试，建立 Wafer Map（晶圆映射图）。合格的芯片划片分离后进入封装环节——与光纤阵列、EIC、热沉等组装为完整的光电系统。

将以上步骤汇总，完整有源平台的工艺序列为：

```text
FEOL（前道工艺）
  SOI 晶圆准备
→ 硅波导 3 道蚀刻（浅 / 深 / 全）
→ [SiN 波导沉积与图形化，可选]
→ 离子注入 6–8 道 + 退火激活
→ Ge 选择性外延 + CMP + 注入 + 退火

BEOL（后道工艺）
→ 上包层沉积
→ 接触孔（W 插塞 + 硅化物）
→ 加热器金属（TiN）
→ M1 / M2 金属互连
→ 钝化 + 焊盘 / 光栅耦合器开窗
→ [深沟槽，边缘耦合可选]
→ 晶圆级测试
→ 划片 → 封装 → 最终测试
```

### 8.5 工艺误差如何影响 PIC

| 工艺变化 | 物理影响 | 器件后果 |
|---|---|---|
| 波导宽度误差 | 有效折射率变化 | 微环 / MZM / AWG 波长漂移 |
| 刻蚀深度误差 | 模式分布变化 | 耦合损耗、弯曲损耗、滤波响应改变 |
| 侧壁粗糙 | 散射损耗增加 | 传输损耗和 Q 值变差 |
| 掺杂波动 | 载流子效应、电阻、电容变化 | 调制效率、损耗、带宽变化 |
| 微加热器 / 金属波动 | 热阻和电流路径变化 | 调谐效率、热串扰、可靠性变化 |
| Ge PD 工艺波动 | 吸收、暗电流、电容变化 | 响应度、噪声、带宽和饱和功率改变 |

## 9. 封装与系统集成：光、电、热一起对准

硅光 PIC 的封装同时处理光接口、电接口和热接口。PIC 侧需要明确封装所需的光、电、热条件：耦合器位置和间距、光纤阵列容差、射频焊盘和微凸点映射、热沉路径、温控范围和测试通路。

### 9.1 光纤耦合和对准

光纤耦合需要定义耦合器类型、光纤间距、主动 / 被动对准、插入损耗、回波损耗、偏振相关损耗（PDL）、温度漂移、光学胶（epoxy）、材料老化和量产一致性。边缘耦合通常给出较好带宽和低损耗潜力；光栅耦合通常给出更好的晶圆级测试便利性。

### 9.2 高速电封装

高速调制器与光电探测器的封装互连需重点进行射频完整性协同设计。片上射频走线、阻抗匹配网络、引线键合（Wire Bonding）或倒装互连引入的寄生电容与电感、射频地回路（Ground Return）的完整性，以及驱动器和跨阻放大器（TIA）与光芯片的物理距离——这些因素共同决定了眼图质量、发射机色散眼图闭合代价（TDECQ）、系统误码率（BER）与接收灵敏度。

### 9.3 热设计

硅光 PIC 对温度敏感，尤其是微环、MZM、AWG 和大量使用微加热器的电路。热设计要考虑散热器、冷板、热电制冷器（TEC）、微加热器功耗、热串扰、环境温度、启动校准和控制算法。

### 9.4 与电子芯片集成

PIC 和 EIC 可以并排放在同一基板上，也可以通过倒装互连、2.5D 中介层、共封装光学或更深度的电子-光子集成结合。集成越紧，寄生越低，热、良率、测试和返修压力也越集中。

## 10. 测试指标：怎样判断 PIC 是否工作正常

测试指标要按链路功能分组，单个最佳值无法代表一颗 PIC 的产品状态。

### 10.1 光学指标

| 指标 | 说明 |
|---|---|
| 插入损耗 / 传输损耗 | 光在耦合器、波导、分束器、滤波器和路径中损失多少 |
| 回波损耗 | 接口或周期结构把多少光反射回去 |
| 耦合损耗 | 光纤或封装光路与 PIC 的模式转换损耗 |
| 信道间串扰 | 相邻波导、滤波通道或波长信道之间的串扰 |
| FSR / Q factor | 微环或滤波器的通道间隔与选择性 |
| 通带 / 阻带 | AWG、微环滤波器或光栅的通过波段和抑制波段 |
| PDL | 偏振相关损耗 |

### 10.2 调制器指标

MZM 和微环调制器通常关注 VπL、消光比、插入损耗、3 dB 电光带宽、每比特能耗、啁啾、线性度、OMA、眼图质量和 TDECQ。指标要与 EIC 驱动器、封装射频路径和目标调制格式一起解释。

### 10.3 探测器指标

Ge PD 通常关注响应度、暗电流、带宽、饱和输入光功率、噪声、电容和线性范围。它们会进入 TIA 噪声、接收灵敏度和封装寄生参数预算。

### 10.4 系统指标

系统侧通常关注链路预算、信噪比（SNR）、BER、前向纠错前误码率（pre-FEC BER）、OMA、TDECQ、功耗、温度范围、可靠性和良率。PIC 的局部指标最终要在这些系统指标中得到验证。

### 10.5 晶圆级测试和封装后测试

晶圆级测试可以通过光栅耦合器、回环测试结构、剪切测试结构、微环监测结构和电焊盘提前建立晶圆映射图（Wafer Map）。封装后测试再验证真实光纤耦合、热稳定性、高速眼图、BER 和控制环路。测试数据会反向反馈给 PDK、版图规则、工艺窗口和分档策略。

## 11. 应用场景：从数据通信到传感和计算互连

硅光 PIC 广泛应用于多种模块形态与系统架构中，其中数据中心高速光互连是其最为核心和最具代表性的商业化应用场景。

| 应用 | PIC 承担的功能 |
|---|---|
| 数据中心光互连 | 把电信号调制到光上，通过光纤传输，再由 PD / TIA 读回 |
| 共封装光学（CPO） | 把光引擎靠近交换芯片或计算芯片，缩短高速电互连 |
| AI / HPC 互连 | 在高带宽和低能耗压力下提供短距到中距光连接 |
| 激光雷达 / 光学相控阵 | 利用相位控制和片上光路实现光束扫描 |
| 传感 | 用微环、MZM 或光谱滤波器感知折射率、温度、生物化学或气体变化 |
| 量子光子学 | 集成干涉器、相位控制器、分束器和探测接口，构建部分量子光路 |

这些应用共享同一套底层语言：材料栈、波导模式、耦合器、无源器件、有源调制、探测、电控制、封装和测试。

## 12. 总结：光信号在硅光 PIC 中的传播路径

PIC 在光引擎中承担两个方向的信号处理。

发送方向（TX）：激光源产生的连续光经耦合器进入芯片，波导将光分配到各通道；调制器接收 EIC 送来的高速电信号，将其写入光载波；被调制后的光信号经输出耦合器送入光纤，离开芯片。

接收方向（RX）：光纤中的光信号经输入耦合器进入芯片，波导和滤波器完成解复用和路由后，Ge PD 将光信号重新变为电信号，由 TIA 放大后交给 EIC。

微加热器和监测光电二极管贯穿两条链路——它们负责校准工艺和温度引起的偏差，稳定调制器的偏置点和微环的共振波长。封装最终将光纤、电极、热沉、驱动器、TIA 和控制系统集成为一个可用的光电模块。

```text
发送方向（TX）：激光源 → 耦合器 → 波导 → 分束器 → 调制器 → 输出耦合器 → 光纤
                          ↑
                    EIC 驱动电路

接收方向（RX）：光纤 → 输入耦合器 → 波导 → 滤波器 / 解复用器 → 光电探测器 → TIA → EIC

控制环路：监测 PD → 控制算法 → 微加热器 → 器件校准（反馈贯穿始终）
```
