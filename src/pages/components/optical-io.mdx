## Optical I/O 是接口层

<TermNote label="Optical I/O" note="optical input/output，光输入输出接口。在光引擎中，它指芯片光路与封装、光纤、连接器之间的光学接口，覆盖 coupler、FAU、lens、connector 和相关测试边界。" /> 把 PIC 波导里的光学模式交给 fiber array、connector、package optical waveguide 或外置激光路径。它的主问题是接口能否低损耗、低反射、偏振可控、可装配、可检查并长期稳定。

这层接口与 [PIC / optical circuit](../pic/) 相邻。PIC 负责在芯片内路由、分光、调制、复用和探测；optical I/O 负责把片上光路交给封装和系统光路。光源见 [激光源](../laser-source/)；die placement、散热、应力和返修见 [Package integration](../packaging/)；光耦合测试、KGD 和封装后验证见 [Manufacturing and test](../manufacturing-test/)。

```text
PIC waveguide
→ spot-size converter / edge coupler or grating coupler
→ lens / fiber array / package optical path
→ ferrule / connector / pigtail
→ system fiber path
```

相关基础页：[Waveguides and optical modes](../../learn/waveguides-and-optical-modes/)、[Refractive index and total internal reflection](../../learn/refractive-index-and-total-internal-reflection/)、[Interference, resonance, and loss](../../learn/interference-resonance-and-loss/)、[Why photonic packaging is hard](../../learn/why-photonic-packaging-is-hard/)。

## 模式转换是核心问题

光纤耦合首先是 <TermNote label="光学模式" note="optical mode，满足波导或光纤边界条件的稳定光场分布。模式决定光场大小、形状、偏振和传播常数。" /> 的匹配问题。PIC 波导模式通常很小，并且由高折射率差强约束；标准通信单模光纤的 <TermNote label="MFD" note="mode-field diameter，模场直径。它描述单模光纤中主模式光场的有效直径，比几何芯径更能反映耦合时看到的光场大小。" /> 通常在 9-10 um 量级。两者直接相接会产生模式失配、反射、散射和对准敏感性。

| 差异 | PIC 波导 | 单模光纤 | 接口后果 |
|---|---|---|---|
| 模式尺寸 | 微米以下到少数微米，强约束 | MFD 约 9-10 um，弱约束 | 需要 taper、spot-size converter、SiN / polymer waveguide 或 lens |
| 模式形状 | 椭圆或矩形近似，TE / TM 差异明显 | 近似圆形高斯分布 | 需要横向、纵向和偏振匹配 |
| 折射率边界 | silicon、SiN、oxide、polymer 等材料突变 | glass core / cladding 弱折射率差 | 界面会带来 Fresnel reflection 和散射 |
| 装配坐标 | die 边缘或表面耦合区 | FAU、ferrule、lens 固定 | 需要 pitch、角度、Z 高度和对准容差 |

模式越小，横向偏移、角度误差和间隙变化造成的损耗越大；模式越大，对准窗口通常更宽，也会增加器件长度、串扰、基底泄漏或封装空间。Optical I/O 的设计就是在低插损、宽带宽、低 PDL、宽装配窗口和高通道密度之间取平衡。

## Coupling 方案怎样选择

常见路线包括 edge coupling、grating coupling 和 lens / package optical path。它们解决的是不同系统约束。

| 路线 | 适合的场景 | 主要约束 |
|---|---|---|
| Edge coupler | 宽带、低插损、高功率输入、WDM 或外置供光路径 | 需要高质量芯片边缘、FAU / lens 对准空间、端面处理和边缘 keepout |
| Grating coupler | wafer-level test、表面 probing、早期 PDK 表征、垂直耦合封装 | 带宽、偏振、反射、方向性和上方散热空间 |
| Lens-assisted path | 改变光束尺寸或方向，放大 working distance，连接封装光路 | 组件数量、焦距误差、表面反射、胶水定位和热漂移 |
| Package optical waveguide / RDL | 把高密度 PIC 光口重新分配到封装边缘或连接器 | 材料损耗、热稳定、工艺成熟度和装配窗口 |

**Edge coupler（边缘耦合器）** 的核心结构是反向锥形（inverse taper）——硅波导尖端缩窄到 &lt;180 nm，光场因波导太窄而向外膨胀进入周围 SiO₂ 或 SiN 包层；随后一个 SiN 波导或聚合物辅助层接住膨胀后的模式，沿 200–600 μm 长度渐进扩束到与单模光纤的模场匹配。这种绝热过渡天然是宽带的（>100 nm 1dB 带宽），典型插损 0.5–1.5 dB/端面，最优设计（SiN SSC + 折射率匹配）可做到 0.2 dB 以下。多尖端设计（2–10-tip）进一步放宽了对准容差和偏振相关损耗。代价是需要高质量芯片端面（深沟槽刻蚀或解理）、精密对准（直接耦合 1dB 横向容差仅 ±1–2.5 μm）、FAU 和光学胶配合，封装成本高于光栅方案。

**Grating coupler（光栅耦合器）** 是在硅波导自身上刻出周期性凹槽（利用浅刻蚀 ~70 nm，周期 600–1800 nm），光沿波导传播时每遇凹槽就有一小部分被向上散射，所有散射在 8–15° 角度上相位叠加形成衍射光束，由上方光纤接收。其核心优势是晶圆级测试——光纤从芯片上方即可对准，无需切割和端面加工。代价是性能：标准光栅的插损约 3–5 dB，1dB 带宽约 30–40 nm，对偏振敏感，标准结构因布拉格对称性存在 >5% 的后向反射。高级设计（切趾光栅、闪耀光栅、双层多晶硅覆盖光栅）可将插损压到 &lt;1 dB、带宽扩展到 >100 nm、后向反射压到 &lt;1%，但增加了工艺复杂度。

**行业现状。** 标准做法是两者分工——晶圆级测试用光栅耦合器从上方快速筛选，最终组装用边缘耦合器满足链路预算。对于 OMA 预算仅约 5–6 dB 的典型 CPO 链路，光栅耦合器在最终组装中不具备可行性（发送+接收合计 6–10 dB 就已超预算）。近年趋势包括混合 GC+EC 结构（同一位置兼具测试与组装功能）、SiN 边缘耦合器（sub-1 dB 损耗成为 CPO 标准）和扩束微透镜（将对准容差放宽到 >±6 μm）。

## 装配容差怎样决定良率

实验室里单通道低损耗耦合，只能证明最佳位置存在。量产要证明的是多通道可以在可接受时间内重复找到位置，并在点胶、固化、热循环和运输后保持余量。

| 装配变量 | 影响 |
|---|---|
| x / y / z alignment | 直接决定 coupling loss 和 channel-to-channel variation |
| pitch / yaw / roll | 多通道 FAU 同时对准时尤其关键 |
| active vs. passive alignment | active alignment 精度高但时间长；passive alignment 依赖机械基准和工艺一致性 |
| epoxy / adhesive | 固化收缩、吸湿、CTE 和老化会引起长期漂移 |
| thermal cycling | 改变 fiber attach、lens attach、edge coupler 相对位置 |
| cleanliness | 颗粒、划伤和残留物会增加插损、反射和高功率损伤风险 |

装配窗口通常用 1 dB / 3 dB alignment tolerance 描述，包括横向、纵向、间隙和角度。若窗口太窄，单颗器件指标再好，也会在量产中表现为吞吐低、长尾损耗大和返修困难。CPO 的高 lane count 放大了这个问题：一组 fiber array 中只要少数通道偏离，就可能拖累整颗 optical engine 的分档。

## Fiber array 与 connector 边界

<TermNote label="FAU" note="fiber array unit，光纤阵列单元。它把多根光纤以规定 pitch 固定在玻璃、硅 V-groove 或 ferrule 结构中，使多通道光纤能一起与 PIC 或封装光路对准。" /> 把单个 coupler 的光学设计变成多通道系统接口。它决定 fiber pitch、端面角、光纤高度、平面度、polish quality、epoxy fillet、strain relief 和 connector compatibility。

| 接口对象 | 要定义的条件 |
|---|---|
| fiber array | pitch、end-face angle、planarity、fiber height、V-groove / ferrule 结构 |
| MT ferrule / MPO 类 connector | 插损、return loss、mating repeatability、cleaning access、blind-mate tolerance |
| pigtail | fiber bend radius、strain relief、routing path、可追溯性 |
| package edge | 出纤方向、heat sink / lid / retention keepout、front-panel 或 mid-board 路径 |

CPO 会让多通道 optical I/O 与散热器、retention frame、ELS input、board components 和机箱光纤管理争夺空间。接口定义越晚，越容易出现 PIC 光口、fiber array、package edge 和 connector 无法同时满足的问题。

## 清洁、反射、偏振与运营

Optical I/O 的现场风险常来自接口表面。灰尘、冷凝、清洗残留、划伤、未完全插合和端面几何异常会增加 insertion loss 或 return loss。高功率外置供光路径中，污染物还可能吸收光功率并造成端面永久损伤。

反射来自折射率突变和周期结构。fiber-air、fiber-glue、glue-facet、lens-air、grating、connector end-face 和 PIC facet 都可能把光返回系统。返回光会扰动激光器、增加 RIN、影响 DFB / external laser 稳定性，或让 ring、monitor 和接收链路出现多径误差。常见控制手段包括 angled facet、AR coating、index-matching、倾斜 grating、低反射 connector 和必要的 isolator。

偏振决定光是否进入 PIC 期待的模式。硅光 waveguide、grating coupler、ring、MZM 和 splitter 常对 TE / TM 敏感。外置供光路径会使用 PM fiber、ferrule 方向标记和 PER 指标来保持偏振轴。旋转误差、应力双折射或 fiber routing 变化，会表现为输入功率下降、lane 间不均或调制器工作点异常。

## 外置激光路径：PM fiber、盲插连接器与高功率边界

外置激光源把 CW light 从 ELS / ELSFP 送到 optical engine。这个路径属于 optical I/O、package 和 operations 的共同边界。

```text
ELSFP / external laser source
→ PM fiber / MT ferrule / blind-mate connector
→ package optical path
→ PIC input coupler
→ splitter and modulator array
```

外置供光的好处是 laser 远离 ASIC 热源并可现场更换；代价是连接器、fiber routing、偏振、反射、功率安全和管理状态变得更重要。OIF ELSFP 语境强调低功率路径验证和清洁流程：未知清洁状态的高功率光路应先检查、清洁、低功率验证损耗，再进入工作功率。

## Optical I/O 规格表

Optical I/O 最终要交付可验收的接口条件，coupler 仿真图只是其中一项证据。

| 规格项 | 应说明的内容 |
|---|---|
| coupler type | edge / grating / lens path、支持波段、偏振、PDK 模型和测试结构 |
| loss budget | coupler loss、fiber-to-fiber loss、connector loss、channel variation、老化余量 |
| return loss | facet、grating、connector 和 laser feedback 相关条件 |
| PDL / PER | TE / TM 支持、PM input、ferrule 方向和偏振验收 |
| alignment tolerance | 1 dB / 3 dB 窗口、x/y/z、pitch/yaw/roll、active / passive alignment 策略 |
| fiber / connector geometry | pitch、端面角、planarity、bend radius、blind-mate 和 cleaning access |
| testability | wafer probing、reference coupler、loopback、package loss decomposition |
| reliability | thermal cycling drift、humidity、adhesive aging、mechanical shock 和 fiber pull |

## 与其他页面的边界

Optical I/O 向 PIC 交付 coupler type、pitch、模式尺寸、PDL、return loss 和 test structure；向 packaging 交付 FAU、connector、出纤空间、清洁和机械窗口；向 manufacturing-test 交付 loss decomposition、alignment log 和 package-level test 条件；向 reliability-operations 交付现场检查、清洁、低功率验证和高功率安全边界。
