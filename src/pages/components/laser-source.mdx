## 激光源的器件边界

AI 数据中心光互连需要稳定、足够功率、可长期工作的连续光。理解这束光的来源，需要回到激光源器件本身：光怎样在 InP / III-V 半导体里产生，怎样被波导、腔体和 DFB 光栅稳定下来，最后怎样表现为输出功率、线宽、RIN、SMSR、PCE 和温度稳定性等指标。

这条路径从材料与有源区开始，经过外延结构、波导、腔体和 DFB 光栅，最后落到 CW 激光器芯片、DFB 激光器、EML、MWL 等光源器件的输出能力。当这些光源进入 CPO、硅光引擎或高密度可插拔模块时，封装形态、光源摆放位置和外置供光方式会进一步影响功率、稳定性、反馈容忍度和长期可靠性要求。

光源封装布局可在 [Package integration](../packaging/) 阅读；晶圆级筛选、known-good-die 和 burn-in 属于 [Manufacturing and test](../manufacturing-test/)；寿命衰减、冗余和现场替换属于 [Reliability and operations](../reliability-operations/)。

## 为什么 CPO 会提高光源要求

CPO 的基本动机是把光引擎放到 <TermNote label="switch ASIC" note="数据中心交换机中的网络交换芯片，负责在端口之间转发数据流；CPO 语境下，光引擎靠近它以缩短高速电互连路径。" /> 附近，缩短高速电信号路径。这个系统位置不会改变半导体激光器的基本发光原理，却会把激光源纳入更紧的热、功耗、供光路径和维护边界约束：

- 硅光 / PIC 主要负责传输、路由、调制和复用光，连续光通常需要由 III-V 激光器提供。
- 光引擎靠近高功耗芯片后，激光源需要在更高的局部温度、更紧的功率预算和更严的长期可靠性约束下保持稳定输出。
- 一束连续光进入 PIC 后会被分光、调制、复用并经历耦合损耗，激光器需要提供足够的出射功率和低噪声。
- 高密度光引擎要求多颗光源在波长、阈值、效率和老化速度上保持一致，系统校准、筛选和冗余设计都会受到这些差异影响。

因此，用于 CPO 的激光源需要在输出功率、热稳定性、噪声、波长一致性和寿命裕量上满足更严格的系统约束。激光器如何从材料、外延和芯片工艺中形成，决定了后续封装与系统架构能够获得多少可用余量。

## 工作原理：电流如何变成稳定光

激光源的原理链由几层机制组成：电流把电子和空穴注入有源区，材料体系决定这些载流子能否高效复合发光；自发辐射先产生少量进入波导和腔体的光子，其中只有满足波导限制、腔体反馈或 DFB 光栅选择条件的光场，才能在受激辐射中被持续放大；波导、腔体和 DFB 光栅再把这些光场约束、反馈并筛选成稳定输出。

<ol className="stage-list">
  <li>
    <strong>激光器把电能变成有序的光</strong>
    <span>半导体激光器先把外部电流变成电子和空穴的注入，再把电子空穴复合释放的能量转化为光子。普通自发发光中，光子的方向、相位和波长分布较散；激光器通过增益介质和反馈结构反复放大同一<TermNote label="光学模式" note="optical mode，满足波导或腔体边界条件的一种稳定光场分布；在激光器中，模式决定光的空间分布、传播方向、偏振以及可被反馈选择的波长。" />，使输出具有更强的方向性、相干性和光谱集中度。“有序”指模式被选择和放大，表现为波长更稳定、噪声更可控的连续光。</span>
  </li>
  <li>
    <strong>InP 相关 III-V 材料提供通信波段发光能力</strong>
    <span>1310 nm / 1550 nm 对应约 0.95 eV / 0.80 eV 的光子能量，因此有源区需要形成接近这些能量的跃迁。实际发光波长由材料带隙、量子阱厚度、应变和能带偏移共同决定；InGaAsP / InAlGaAs 等直接带隙 III-V 合金可以通过成分和量子阱设计覆盖通信波段。InP 衬底的重要性在于晶格平台：这些有源区和限制层可以在接近 InP 晶格常数的条件下外延生长，降低位错和非辐射复合风险。</span>
  </li>
  <li>
    <strong>电流通过 p-n / p-i-n 结构注入电子和空穴</strong>
    <span>激光二极管通常通过正向偏置工作：外加电压降低结区势垒，使 n 侧电子和 p 侧空穴被注入到中间的有源区。真实器件常采用异质结和 p-i-n 类结构，把载流子集中到低带隙有源层附近，同时用包层和限制层把光场保持在相同区域。量子阱把电子和空穴限制在纳米级薄层中，并通过能级、应变和光场重叠设计提高材料增益，使电流更有效地转化为可被放大的光。</span>
  </li>
  <li>
    <strong>从发光到激光：增益超过损耗</strong>
    <span>低电流时，有源区已经会发光，但主要是自发辐射。光子从不同位置、沿不同方向、以分散的相位和波长产生，因此只能形成微弱而分散的发光。随着电流升高，注入有源区的电子和空穴增多，材料开始对经过它的光提供增益：已有光子可以诱导新的电子空穴复合，产生与原光子频率、相位、偏振和传播方向一致的新光子。波导、腔体或 DFB 光栅会保留满足传播和反馈条件的那部分光场；其他方向、波长或相位不匹配的光很快衰减。当被保留下来的光在一轮传播中获得的增益足以补偿吸收、散射和输出耦合等损耗时，器件达到激光阈值。超过阈值后，受激辐射成为主导，输出功率快速上升，激光器进入稳定工作状态。</span>
  </li>
  <li>
    <strong>波导、腔体和光栅共同选择可放大的光</strong>
    <span>光在芯片里以光学模式沿波导传播。垂直方向上，包层、SCH 和有源区通过折射率差把光场限制在有源区附近；横向方向上，ridge waveguide 或 buried heterostructure 控制模式尺寸，并让主要光场与电流注入区域重叠；纵向方向上，端面反射、DBR 或 DFB 光栅提供反馈。波导决定光场在截面中的分布，腔体和光栅决定哪些波长和纵向模式能在往返传播中保留下来，三者共同决定哪些光能被持续放大。</span>
  </li>
  <li>
    <strong>DFB 光栅选择稳定单一波长</strong>
    <span>DFB 的核心是把周期性光栅放在波导附近，让光在传播过程中持续遇到微小的折射率扰动。光栅周期和波导有效折射率共同决定 Bragg 波长；接近这个波长的前向光和后向光会被强烈耦合，从而沿整个腔长获得分布式反馈。光栅相移、耦合强度和腔长设计会进一步提高目标纵模相对边模的阈值优势，使输出更接近稳定单一波长。SMSR、线宽、中心波长漂移和反射敏感性就是评价这种单模稳定性的关键指标。</span>
  </li>
  <li>
    <strong>真实芯片还要通过热、噪声、反射和一致性考验</strong>
    <span>把原理做成量产芯片时，稳定性取决于多个约束。高功率 CW 工作会增加结温，使带隙、折射率、增益峰和腔模位置随温度移动；载流子和光子数量的涨落会表现为强度噪声，腔内自发辐射噪声和线宽增强效应会影响线宽；外部反射光返回腔体时，还可能扰动单模输出。材料缺陷、端面质量、金属接触、散热路径、片内一致性和老化筛选，则决定激光器能否长期维持足够的输出功率和光谱稳定性。</span>
  </li>
</ol>

## 制造路径：从 InP 衬底到激光器芯片

器件制造可以沿六个阶段理解：每一阶段既包含工艺动作，也对应形成的关键激光器结构。主线停在激光器芯片和器件级输出接口，系统级封装位置详见封装页面。

  <li>
    <strong>InP 单晶生长与衬底制备</strong>
    <span>从高纯 In / P 原料制备多晶 InP，再生长单晶晶锭，并经过切片、研磨、抛光和 CMP，得到 epi-ready InP 衬底。</span>
  </li>
  <li>
    <strong>外延生长</strong>

  </li>
  <li>
    <strong>光栅与波导加工</strong>
    <span>通过光刻、刻蚀、再生长或相关工艺形成 DFB 光栅、ridge waveguide 或 buried heterostructure，用来限制光场并提供反馈。</span>
  </li>
  <li>
    <strong>电极、绝缘与电流限制</strong>
    <span>形成 p/n contact、金属电极、钝化层和电流限制结构，使载流子能够被控制地注入有源区。</span>
  </li>
  <li>
    <strong>解理与端面处理</strong>
    <span>把晶圆加工成 laser bar / laser chip，并进行端面镀膜，控制反射、输出耦合和端面可靠性。</span>

  <li>
    <strong>芯片测试与筛选</strong>
    <span>测试阈值电流、输出功率、光谱、SMSR、RIN 和老化表现，筛选可进入后续组装的激光器芯片。</span>

## 1. InP 单晶生长与衬底制备

面向 1310 nm / 1550 nm 通信波段的半导体激光器，常建立在 InP 相关 <TermNote label="III-V 材料体系" note="由元素周期表 III 族和 V 族元素组成的化合物半导体体系，例如 InP、GaAs、InGaAsP、InAlGaAs。它们常用于高效发光、探测和高速光电子器件。" /> 上。这个选择来自目标波长、<TermNote label="有源区" note="激光器中真正产生光增益的区域。电子和空穴在这里复合，释放光子；通信激光器通常把有源区设计成量子阱或多量子阱。" />所用材料和<TermNote label="外延晶格" note="外延是在晶体衬底表面继续生长晶体薄层的工艺。新生长的材料会沿着衬底的晶格排列，因此外延材料与衬底的晶格匹配会影响缺陷和应变。" />之间的共同约束：有源区需要在通信波段形成光增益，外延层需要在低缺陷晶格模板上连续生长，InP 衬底则提供了承载这些 III-V 结构的起始平台。

### 为什么需要 InP 衬底

激光器的最低层原理是电子能量转换。半导体里，电子可以被注入或激发到较高能量状态；当电子回到较低能量状态，并与空穴复合时，会释放能量。若这部分能量以光子的形式释放，就产生光；若在受激辐射过程中产生的新光子与已有光子具有相同频率、相位和传播方向，光就可以被放大，形成激光。

通信波段激光器的材料选择，首先受目标光子能量约束。光子能量和波长满足近似关系 `E(eV) = 1.24 / λ(um)`，所以 1310 nm 约对应 0.95 eV，1550 nm 约对应 0.80 eV。激光器的有源区需要具有接近这些能量的 <TermNote label="带隙" note="半导体中价带顶和导带底之间的能量差。电子从导带回到价带并与空穴复合时，释放出的能量大致由带隙决定。" />，并且要能在电注入后高效产生辐射复合和光增益。

对发光器件来说，还要看这种复合是否容易把能量交给光子。<TermNote label="直接带隙" note="导带底和价带顶位于相近的晶体动量状态，电子和空穴复合时更容易直接放出光子，因此更适合高效发光。" />材料更适合做激光器有源区；<TermNote label="间接带隙" note="导带底和价带顶不在相同动量位置，复合时通常还需要声子参与，能量更容易变成晶格振动和热，发光效率较低。" />材料则更难作为高效原生光源。InP 相关 III-V 材料体系的意义在于，它提供了适合高效发光的直接带隙材料，并且可以通过调整 InGaAsP / InAlGaAs 等合金的元素比例，把有源区带隙调到 1310 nm / 1550 nm 通信波段所需的光子能量附近。

纯 InP 的室温带隙约 1.34 eV，对应更短波长。通信波段激光器的主要光增益通常来自 InGaAsP 或 InAlGaAs <TermNote label="量子阱" note="夹在较大带隙材料之间的极薄半导体层，可以把电子和空穴限制在很窄的区域内，提高复合概率和光增益效率。" /> / 多量子阱。选择 InP 衬底的核心原因在于平台匹配：这些有源区材料可以通过成分设计把带隙调到通信波段，同时让晶格常数接近 InP，从而在 InP 上形成低缺陷外延结构。

这一材料选择可以从三个层面理解：

- **带隙层面**：有源区材料的带隙决定可产生的光子能量，因而决定目标波长附近能否形成增益峰。
- **晶格层面**：有源区、限制层和包层需要沿衬底晶格连续生长，晶格常数接近时，应变和位错更容易控制。
- **器件层面**：InP 平台可以承载 InP cladding、SCH、InGaAsP / InAlGaAs QW、接触层、DFB 光栅和波导加工，形成成熟的通信波段激光器工艺链。

### 晶格匹配为什么重要

从晶体物理看，衬底提供了外延生长的原子排列周期模板。所谓 <TermNote label="晶体" note="原子或离子按照周期性规则排列形成的固体。半导体晶体中的周期结构决定了能带、晶向、缺陷和外延生长方式。" />，是原子在三维空间里按照固定周期重复排列形成的固体。描述这个重复单元尺寸的关键参数之一，就是 <TermNote label="晶格常数" note="晶体周期结构中的基本长度尺度，可以理解为原子排列的节距。外延层和衬底晶格常数接近时，应变和位错更容易控制。" />。InP 采用 zincblende 晶体结构，晶格常数约为 5.87 A。

外延生长时，新沉积的 III-V 材料会沿着衬底表面的晶格继续排列；如果外延层的本征晶格常数接近 InP，就能以较低应变长成连续薄层。这个过程可以理解为“沿着已有晶体模板继续排队”：模板越平整、周期越匹配，后续薄层越容易保持同样的晶体秩序。

InGaAsP 和 InAlGaAs 这类多元 III-V 合金的价值在于带隙和晶格常数可以协同设计：一方面通过成分比例调节带隙，使电子空穴复合释放的光子能量对应 1310 nm / 1550 nm 附近的波长；另一方面让材料晶格常数接近 InP，使有源区、限制层和包层能够沿 InP 晶格模板生长。对激光器来说，带隙决定光增益能否位于目标波段，晶格匹配决定这个有源区能否以低缺陷方式长出来。

如果外延层和衬底之间存在较大 <TermNote label="晶格失配" note="外延层本征晶格常数与衬底不同，生长时会产生应变。常规量子阱 / DFB 激光器通常要控制失配以避免位错；但在 Stranski-Krastanov 自组装量子点结构中，受控失配应变也可促使二维润湿层转为三维量子点 islands，部分量子点激光器会利用这一机制。" />，薄层会先被拉伸或压缩。应变超过材料能够承受的范围后，晶体会用 misfit dislocation、threading dislocation 等缺陷释放应变。这些缺陷会穿过有源区或靠近有源区，成为非辐射复合中心、漏电通道和长期失效起点。载流子本应在 QW / MQW 有源区里复合并贡献光增益，缺陷会把一部分能量转化为热和损耗，表现为阈值电流升高、斜率效率下降、暗电流增大和可靠性变差。

因此，InP 衬底这一层决定的变量很多：晶格模板、晶向、缺陷底噪、表面粗糙度、污染水平、翘曲和厚度一致性。后续外延、光刻、解理和测试都会把这些起点条件继续放大。

### InP 衬底怎样制造

InP 衬底制造的目标，是把元素级 In / P 原料转化为化学计量、晶向、缺陷密度和表面质量都可控的 InP 晶圆。

第一步是高压合成多晶 InP。磷属于高蒸气压 Group V 元素，温度升高后容易挥发；合成时需要高压环境、受控温场和能够承受反应条件的反应器。典型设备包括高压石英玻璃反应器、分区加热系统、惰性气体 / 压力控制、热电偶和装载铟的坩埚。工艺上，磷先在较低温区升华成气相，再输运到液态铟附近并反应形成 InP。反应完成后，通过定向凝固和蚀刻降低非化学计量成分、夹杂和残余污染，为后续单晶生长准备原料。

第二步是单晶生长。VGF（Vertical Gradient Freeze）等工艺把多晶 InP 重新熔化，并在受控温度梯度下推进固液界面，使晶体沿种晶方向长成单晶晶锭。典型 VGF 系统会把预合成 InP 放入 <TermNote label="pBN 坩埚" note="pyrolytic boron nitride，热解氮化硼坩埚，常用于高温化合物半导体生长，具有较高化学稳定性并可降低对熔体的污染。" />，再放入多区控温炉中；炉体通过不同加热区形成熔点上下的近恒温区和较小的轴向温度梯度。晶体通常从定向种晶开始，固液界面向上推进。关键控制量包括温度梯度、界面形状、生长速率、掺杂均匀性和热应力。温度场越不稳定，越容易引入位错、应力和晶向偏差；这些问题会传递到后续外延层。

第三步是晶圆加工。单晶晶锭需要先定向，确定晶向和切片角度；随后通过内圆切割或线切割把晶锭切成晶片。线切割会用携带磨料的线穿过晶体，效率高，但也会引入切割损伤和表面应力。切片后还要经过边缘倒角、表面研磨、刻蚀、抛光和 CMP，把晶片做成厚度、平坦度、粗糙度都满足外延要求的 wafer。

第四步是清洗、检测和分级。清洗线需要去除颗粒、有机 / 无机薄膜和残余抛光液，并重新控制表面氧化层状态。检测环节会用高分辨率显微镜检查可见缺陷，用腐蚀坑密度 <TermNote label="EPD" note="etch pit density，腐蚀坑密度。对 InP 可用含 HBr 的腐蚀液显露位错穿出表面的位置，再通过扫描计数来评估晶体位错密度。" /> 评估位错，用 Hall / van der Pauw 测量电学参数，也可用非接触电阻率映射观察晶圆内均匀性。达到外延要求后，这片晶圆才可以称为 epi-ready InP substrate。

从设备角度看，InP 衬底制造可以看成一条从高压化学合成到精密表面准备的工具链：

| 环节 | 典型器材或工具 | 作用 |
|---|---|---|
| 高压合成 | 高压石英反应器、分区加热系统、气压控制、热电偶、坩埚 | 让高纯 In / P 在可控压力和温度下形成多晶 InP，并控制磷挥发。 |
| VGF 长晶 | pBN 坩埚、多区控温炉、种晶、温度梯度控制系统 | 控制固液界面、晶向和热应力，把多晶料转化为低位错单晶晶锭。 |
| 定向与切片 | X-ray 定向、内圆切割或线切割设备 | 确定晶向和切片角度，把晶锭加工成晶片。 |
| 研磨与 CMP | 研磨盘、抛光垫、化学机械抛光液、双面 / 单面抛光工具 | 去除切割损伤，控制厚度、平坦度、粗糙度和表面损伤层。 |
| 清洗与检测 | 超声 / 化学清洗、显微镜、EPD 扫描、Hall / van der Pauw、非接触电阻率映射 | 控制颗粒、污染、位错、电学均匀性和 epi-ready 表面状态。 |

| 工艺关注点 | 对激光源的影响 |
|---|---|
| 原料纯度 | 金属杂质和残余污染会缩短载流子寿命，增加暗电流和早期失效概率。 |
| 化学计量 | In 与 P 的比例和 Group V 元素挥发需要受控，避免形成富 In / 富 P 缺陷和夹杂。 |
| 单晶质量 | 位错密度、夹杂和微裂纹会影响外延连续性，并可能成为器件失效起点。 |
| 几何一致性 | 晶圆厚度、TTV、bow 和 warp 会影响外延均匀性、光刻焦深、解理方向和后续测试稳定性。 |

## 2. 外延生长

外延生长把上一节得到的 epi-ready InP 衬底变成激光器的功能材料栈。衬底提供晶体模板，外延工艺在这个模板上逐层生长 III-V 薄层：有些层负责导电，有些层负责限制光场，有些层负责把电子和空穴限制在很薄的区域内并产生光增益。对半导体激光器来说，外延质量基本决定了后续器件加工能够达到的性能上限。

### 外延的物理含义

外延可以理解为“沿着已有晶体模板继续排列”。InP 衬底表面已经具有周期性的原子排列；当 In、Ga、Al、As、P 等原子到达加热后的表面时，它们会在合适位置吸附、迁移并并入晶格，形成新的单晶薄层。每一层的材料成分、厚度、掺杂和界面质量，都由反应气体、温度、压力和生长时间共同决定。

这一阶段要同时解决三个问题：

- **材料成分**：InGaAsP / InAlGaAs 的组分决定带隙，进而决定有源区能否在 1310 nm / 1550 nm 附近提供光增益。
- **晶体连续性**：外延层需要沿 InP 晶格模板低缺陷生长，避免位错、界面粗糙和应变失控。
- **空间分工**：激光器需要把 <TermNote label="载流子" note="在半导体中能够携带电流的粒子，主要指电子和空穴。激光器里，电子从 n 侧注入、空穴从 p 侧注入，并在有源区复合产生光增益。" /> 限制在有源区，把光场限制在 <TermNote label="波导" note="引导光在芯片内沿特定路径传播的结构。激光器里，波导通过折射率差和几何边界把光场限制在有源区附近，提高光场与增益区的重叠。" /> 附近，同时还要提供低阻电流注入路径。

### MOCVD / MBE 的生长过程

量产 III-V 光电子器件常使用 MOCVD（Metal-Organic Chemical Vapor Deposition，金属有机化学气相沉积）。MOCVD 把含 III 族元素的金属有机前驱体和含 V 族元素的气体送入反应腔，在加热晶圆表面分解并形成单晶外延层。反应器通常包含气体混合与输运系统、金属有机前驱体源瓶、氢气或氮气载气、加热基座、多区温控、压力控制、尾气处理和晶圆旋转结构。前驱体气体流过加热的 InP 晶圆表面后分解，原子并入晶格；改变前驱体比例可以改变 In/Ga/Al/As/P 的相对含量，改变掺杂气体可以形成 p 型或 n 型层，改变生长时间则控制薄层厚度。

MBE（Molecular Beam Epitaxy，分子束外延）提供另一条路线：在超高真空中用元素或分子束照射晶圆，并通过独立源炉、快门和束流控制 In、Ga、Al、As、P 等材料供给。MBE 的优势是界面切换清晰、厚度控制精细；MOCVD 的优势是产能、均匀性和产业成熟度更适合大规模化合物半导体制造。

两种设备路线的差异可以概括为：MOCVD 主要通过气体流量、V/III 比、温度、压力和旋转均匀性来控制外延；MBE 主要通过超高真空、源炉温度、快门切换、束流监测和 RHEED 原位观察来控制外延。

```text
epi-ready InP substrate — 外延起始表面
→ MOCVD / MBE epitaxy — 外延生长多层半导体结构
→ cladding / SCH / MQW / contact layers — 形成包层、限制层、有源区和接触层
```

### 激光器外延层的功能分工

一颗边发射 InP 激光器的外延结构通常由一组功能层构成。可以从下往上粗略理解为：

| 层级 | 常见材料或结构 | 作用 |
|---|---|---|
| InP 衬底（InP substrate） | n-InP 衬底 | 提供晶格模板、机械支撑和下侧电流路径。 |
| 下包层（lower cladding） | n-InP 或相关 InP 层 | 通过较低折射率把光场限制在中间波导附近，并承担 n 侧电流注入。 |
| 分离限制层（SCH / optical confinement layer） | InGaAsP / InAlGaAs 限制层 | <TermNote label="SCH" note="Separate Confinement Heterostructure，分离限制异质结构。它把极薄量子阱放在较宽的波导/限制层中，使载流子主要留在量子阱，光场则分布在更宽的波导区域。" /> 用来改善光场和载流子的空间重叠。 |
| 量子阱有源区（QW / MQW active region） | InGaAsP 或 InAlGaAs 量子阱 / 多量子阱 | 电子和空穴在该区域复合并产生光增益，是决定波长、阈值和效率的核心区域。 |
| 上包层（upper cladding） | p-InP 或相关 InP 层 | 形成上包层，限制光场并提供 p 侧电流注入路径。 |
| 接触层（contact layer） | 高掺杂 InGaAs / InGaAsP 等 | 降低金属接触电阻，为后续电极加工提供界面。 |

<TermNote label="量子阱" note="quantum well，夹在较大带隙材料之间的极薄低带隙半导体层。电子和空穴在厚度方向被限制，复合概率提高，光增益和阈值电流更容易被工程化控制。" />厚度通常只有纳米量级。多个量子阱叠在一起形成 MQW，可以在保持载流子限制的同时增加总增益。量子阱之间的 barrier 层负责隔开各个阱，并帮助调节应变、载流子分布和光学模式重叠。

Optical confinement layer 指外延材料栈里的垂直光场限制区域：光场主要由包层、SCH / waveguide core 和有源区共同限定。ridge waveguide、buried heterostructure 等横向波导结构通常还需要后续光刻、刻蚀、再生长或钝化工艺形成，属于器件加工环节。

### 外延层怎样决定激光源性能

外延结构把“材料能发光”变成“器件可稳定输出”。其影响可以分成几类：

| 外延设计项 | 对激光源的影响 |
|---|---|
| 有源区带隙 | 决定光增益峰值落在哪个波段，并影响温度漂移。 |
| 量子阱厚度与数量 | 影响阈值电流、微分增益、饱和输出功率和高温效率。 |
| SCH 与包层折射率 | 决定光场限制、模式尺寸、内部损耗和与后续波导结构的匹配。 |
| 掺杂分布 | 影响串联电阻、电流扩展、自由载流子吸收和局部发热。 |
| 应变与晶格匹配 | 影响位错形成、可靠性和长期老化。 |
| 界面粗糙与厚度误差 | 造成散射损耗、增益不均匀和波长偏移。 |

通信波段常用 InGaAsP / InAlGaAs 等有源区材料覆盖 1310 nm 和 1550 nm 附近窗口。InAlGaAs/InP 体系具有较大的导带偏移，有助于在高温下限制电子泄漏；InGaAsP/InP 工艺成熟，常用于通信波段 DFB 和 EML 结构。实际选择通常取决于目标波长、输出功率、温度范围、阈值、效率、可靠性和制造成熟度。

### 工艺控制点

外延看起来是在“长材料”，实际控制对象是一组相互耦合的物理变量：

| 控制对象 | 典型控制方式 | 影响 |
|---|---|---|
| 成分比例 | 调节 III 族 / V 族前驱体流量、V/III 比和生长温度。 | 决定带隙、晶格常数、应变和折射率。 |
| 厚度 | 控制生长速率和生长时间，也可结合原位监测校准。 | 决定量子阱能级、波导模式和 DFB 目标波长的基础偏差。 |
| 掺杂 | 引入 Zn、Si、S 等掺杂源并控制浓度剖面。 | 决定 p/n 区导电性、接触电阻、吸收损耗和热负载。 |
| 界面质量 | 控制气体切换、温度稳定、反应腔记忆效应和表面洁净度。 | 决定非辐射复合、散射损耗和器件一致性。 |
| 晶圆均匀性 | 优化气流、温场、压力、晶圆旋转和装片方式。 | 决定同一片晶圆上阈值、波长和输出功率的一致性。 |

### 哪些外延细节会成为工艺壁垒

外延层的公开知识和量产配方属于不同层级。公开资料通常能够说明 InP 激光器为什么采用 InGaAsP / InAlGaAs 有源区、为什么需要 QW / MQW、SCH 和包层，也能看到一些论文或讲义给出示例结构、波导设计、量子阱数量、腔长、内部损耗和量子效率。这些内容构成行业共同语言，足以理解器件原理和主要设计变量。

进入制造环节后，同样的层名会变成更细的工艺问题：每层的实际厚度、成分比例、应变量、p/n 掺杂浓度和掺杂剖面，需要与具体 MOCVD / MBE 设备、反应腔历史、前驱体纯度、气体切换、温度场、晶圆旋转、装片方式和在线 / 离线计量相互匹配。公开论文中的某个层厚或掺杂值可以作为设计例子，不能直接等同于另一条产线的可复制配方。

InP 激光器厂商的核心壁垒通常由两部分组成：理解器件应该有哪些功能层，以及把目标波长、阈值电流、输出功率、RIN、SMSR、温度漂移和寿命要求，稳定地转化为可量产的外延窗口和筛选规则。后者包括生长速率校准、V/III 比控制、掺杂记忆效应、界面切换、片内均匀性、再生长或钝化兼容性，以及外延数据与后续光刻、解理、测试、老化结果之间的反馈闭环。能长期保持良率和可靠性的参数窗口，往往比单张外延结构示意图更难复制。

外延已经把激光器的波长、阈值、效率、温度敏感性和寿命风险写进了材料栈。当激光源用于 CPO 或其他高密度光互连场景时，高功率连续工作和较高温度稳定性会使有源区设计、限制层设计、掺杂和缺陷控制变得更关键。

## 3. 光栅与波导加工

外延层已经给出垂直方向的材料结构：低折射率包层、高折射率限制层和有源区共同决定光场在厚度方向上的分布。器件加工继续完成另外两个方向的控制：横向上把光场和电流限制在窄条区域内，纵向上用腔体和 DFB 光栅选择稳定波长。

### 波导为什么能限制光

光在半导体芯片里传播时，会优先停留在折射率较高、边界条件允许的区域。这个区域称为波导的 core；周围折射率较低的材料称为 cladding。真实光场会以 <TermNote label="光学模式" note="optical mode，满足波导边界条件的一种稳定电磁场分布。它决定光场中心、扩展范围、偏振和与有源区的重叠。" /> 的形式分布在 core、cladding 和有源区附近，并向几何 core 外侧延伸。

边发射 InP 激光器通常需要同时控制三种约束：

| 约束方向 | 主要结构 | 作用 |
|---|---|---|
| 垂直方向 | 外延包层、SCH、量子阱和波导芯层 | 把光场压在有源区附近，提高光场与增益区的重叠。 |
| 横向方向 | ridge waveguide、buried heterostructure、stripe aperture | 限制光场和电流的横向扩展，降低横向多模和电流扩散。 |
| 纵向方向 | 端面、DBR / DFB 光栅、相位移区域 | 决定反馈、腔长、模式选择和目标波长。 |

这对应两个不同层级：SCH / optical confinement layer 主要来自外延材料栈，解决垂直方向的光场限制；ridge waveguide 和 buried heterostructure 属于器件加工形成的横向结构，解决光场与电流在芯片宽度方向的分布。

### Ridge 和 buried waveguide 怎样加工

Ridge waveguide 的基本做法，是先用光刻定义一条窄脊，再通过刻蚀移除脊两侧的上层材料。脊区保留较高的等效折射率，两侧被刻低或被介质覆盖后，横向等效折射率下降，光学模式便更容易停留在脊下方的有源区附近。这个结构同时会改变电流路径：金属电极通常只在脊上方开窗口接触，电流主要从脊区注入。

Buried heterostructure 的做法更复杂。器件先刻出包含有源区的窄条，再通过再生长把两侧埋入 InP 或电流阻挡层中。被埋入的结构可以同时提供横向折射率约束和电流阻挡，适合低阈值、单横模和较好热稳定性的边发射激光器。代价是再生长界面、侧壁损伤、阻挡层掺杂和洁净度都会进入良率与可靠性问题。

| 结构 | 典型工艺 | 优点 | 需要控制的风险 |
|---|---|---|---|
| ridge waveguide | 光刻、硬掩膜、干法或湿法刻蚀、介质钝化、金属开窗 | 工艺相对直接，便于定义横向模式和电流孔径。 | 侧壁粗糙、刻蚀深度、脊宽和介质应力会影响损耗、阈值和可靠性。 |
| buried heterostructure | 台面刻蚀、表面清洁、InP 或阻挡层再生长、平坦化和接触加工 | 横向光场与电流限制更强，热和模式控制更好。 | 再生长界面缺陷、电流泄漏、阻挡层设计和应力控制更难。 |
| laterally coupled grating | 在有源区两侧或波导邻近区域制作周期结构 | 光栅可避开直接破坏有源区，便于调节耦合强度。 | 光栅与模式重叠较弱时，需要更精确的几何和耦合设计。 |

### DFB 光栅如何选择波长

DFB 是 distributed feedback，指沿光传播方向分布的反馈结构。它依靠一串周期性折射率扰动提供反馈。每一个光栅周期都会产生很弱的反射；当这些微反射在某个波长上相位相加，前向传播的光便会被有效耦合成后向传播的光，形成分布式反馈。

一阶 DFB 光栅的 Bragg wavelength（Bragg 波长）可粗略写成：

```text
lambda_B = 2 n_eff Lambda
```

其中 `lambda_B` 是被选择的中心波长，`n_eff` 是波导模式看到的有效折射率，`Lambda` 是光栅周期。这个关系说明两件事：光栅周期直接决定目标波长，有效折射率也会把材料、温度、波导尺寸和工艺误差带入波长结果。

光栅还要看 <TermNote label="耦合系数" note="grating coupling coefficient，常记作 kappa，表示光栅把前向波耦合到后向波的强度。它由折射率扰动、光栅深度、占空比、长度以及光学模式与光栅的重叠决定。" />。耦合太弱，反馈不足，阈值和边模抑制受影响；耦合太强，光栅损耗和空间烧孔等问题会变得突出。实际 DFB 设计还常加入四分之一波相移或等效相位设计，把两个对称边模中的一个中心模式变成最低阈值模式，从而提高单模稳定性和 SMSR。

### 工艺误差怎样变成器件误差

光栅与波导加工把纳米级几何误差转化为光学误差。对通信激光器来说，这些误差会直接进入波长、阈值、边模抑制、线宽、RIN、损耗和长期可靠性。

| 工艺变量 | 物理影响 | 器件表现 |
|---|---|---|
| 光栅周期 | 改变 Bragg 波长。 | 中心波长偏移，WDM 通道对准变差。 |
| 有效折射率 | 受材料成分、波导尺寸、温度和载流子密度影响。 | 波长随温度和电流漂移，批次一致性变差。 |
| 光栅深度 / 占空比 | 改变耦合系数和散射损耗。 | SMSR、阈值电流、斜率效率和线宽发生变化。 |
| 光栅相位 / 拼接误差 | 扰乱沿腔体的反馈相位关系。 | 单模选择变弱，边模抑制下降。 |
| 波导侧壁粗糙 | 增加散射损耗和局部缺陷。 | 阈值升高，输出功率和可靠性下降。 |
| ridge 宽度 / 刻蚀深度 | 改变横向模式和电流孔径。 | 横向多模、模式跳变或局部发热风险上升。 |
| 再生长界面 | 引入界面缺陷、污染或应力。 | 漏电、非辐射复合和早期失效概率增加。 |

实际使用中，外部反射也会反馈到 DFB 腔体。光源器件需要通过端面镀膜、反馈容忍度设计、较低 RIN 和稳定单模工作来降低这类扰动；后续封装和系统还会继续处理连接器反射、光隔离和耦合回波。

## 4. 电极、绝缘与电流限制

半导体激光器是电流驱动的器件。外部驱动电路提供电流，电极和半导体接触层把这股电流送入 p-n / p-i-n 异质结构，电子和空穴在有源区附近相遇并复合。电极加工的难点在于同时满足低电阻注入、低光学吸收、低局部发热和长期界面稳定性；任一环节失控，都会把外延层已经建立好的增益优势消耗掉。

### 电流怎样进入有源区

一颗边发射激光器的光沿芯片长度方向传播，电流通常沿垂直方向穿过外延层。上方 p-contact 把空穴注入 p 侧，下方 n-contact 或衬底侧金属提供电子注入路径。载流子进入有源区后，受激辐射把电流注入转化为腔内光增益；当增益超过内部损耗、镜面或光栅损耗和输出耦合损耗时，器件越过阈值并进入稳定激光工作状态。

这个过程受三个几何关系控制：

| 几何关系 | 物理含义 | 影响 |
|---|---|---|
| 电流路径与有源区重叠 | 电流是否主要穿过 QW / MQW 区域。 | 决定阈值电流、效率和局部发热。 |
| 电流孔径与光学模式重叠 | 被注入的载流子是否覆盖主要光场区域。 | 决定增益利用率和横向模式稳定性。 |
| 电极 / 绝缘窗口与波导位置 | 金属、介质和刻蚀边界是否破坏光场或增加吸收。 | 决定内损耗、串联电阻、热阻和可靠性。 |

### 欧姆接触和接触层

金属和半导体之间需要形成低电阻接触。激光器外延顶部通常会设计高掺杂 InGaAs 或 InGaAsP 接触层，使金属更容易把电流注入 p 侧。金属堆栈沉积后，常通过退火或合金化降低接触电阻。n 侧通常通过导电衬底背面金属或 n-contact 完成电流回路。

接触电阻过高时，外部电压会更多地消耗在接触和串联电阻上，转化为热而非光输出。局部发热会改变带隙和折射率，使波长漂移、阈值升高、RIN 变差，并加速金属扩散、界面退化和缺陷增长。

### 绝缘、钝化和电流窗口

电极不能覆盖所有暴露半导体表面。工艺上通常先沉积 SiN、SiO2 或类似介质层，对刻蚀后的侧壁和表面进行钝化；随后在需要注入电流的位置开窗口，再沉积金属。介质层承担三项功能：隔离无须导电的区域，保护侧壁和表面，定义电流注入孔径。

钝化质量会影响器件寿命。刻蚀侧壁附近容易有悬挂键、污染和等离子损伤，这些位置会形成漏电通道或非辐射复合中心。介质层覆盖不完整、应力过大或开窗边缘不稳定时，电流会绕开理想路径，产生局部热点和早期失效风险。

### 电流限制结构

| 微观结构 | 说明 |
|---|---|
| p-contact / metal | 连接外部电极，把电流送入 p 侧接触层。 |
| n-contact / back metal | 通过 n 侧外延层或导电衬底提供回流路径。 |
| stripe contact | 只在窄条区域形成金属接触，让电流优先从条形区域进入有源区。 |
| ridge waveguide | 通过刻蚀形成脊形结构，同时限制横向光场和电流扩展。 |
| buried heterostructure | 把有源条埋入再生长材料或阻挡层中，同时提供横向光场和电流限制。 |
| current blocking layer | 通过半绝缘层、反向结或 p-n-n 阻挡结构减少侧向漏电。 |
| dielectric aperture | 通过绝缘介质开窗定义电流注入区域。 |

Stripe contact 工艺简单，但横向电流扩散和光学模式控制有限。Ridge waveguide 把几何刻蚀、电流孔径和横向模式控制结合在一起。Buried heterostructure 的限制能力更强，但需要再生长和阻挡层设计；高温工作时，阻挡层厚度、掺杂和几何尺寸会影响漏电电流。

### 工艺选择怎样影响输出

| 工艺变量 | 对激光源的影响 |
|---|---|
| 接触电阻 | 提高工作电压和焦耳热，降低 PCE，并加重温度漂移。 |
| 金属扩散 / 界面反应 | 可能污染接触层或靠近有源区，导致长期退化。 |
| 绝缘开窗位置 | 决定电流是否对准 ridge 和光学模式中心。 |
| 侧壁钝化 | 影响漏电、表面复合、湿气敏感性和早期失效概率。 |
| 电流孔径宽度 | 太宽容易横向多模，太窄会增加电阻和局部发热。 |
| 阻挡层设计 | 决定 buried heterostructure 的侧向漏电和高温效率。 |
| 热路径 | 影响阈值、电光效率、波长漂移、RIN 和寿命。 |

在激光器截面中，电流路径、光学模式和有源区位置需要同时成立：理想器件让载流子主要进入有光场的区域，并尽量减少旁路漏电、金属吸收、侧壁复合和局部过热。

## 5. 解理与端面处理

晶圆完成器件加工后，需要沿晶体解理面分成 laser bar，再进一步分割成 laser chip。边发射激光器的端面具有双重角色：它是光学腔的一部分，也是光输出窗口。端面质量会直接进入腔体反射、散射损耗、输出耦合和高功率可靠性。

对 Fabry-Perot 激光器来说，两个端面的反射本身就构成主要腔体反馈。对 DFB 激光器来说，分布式光栅提供主要波长选择和反馈，端面反射仍会改变输出方向、输出耦合和外部反馈敏感性。高功率 CW 光源通常会在一端使用较低反射或增透膜，在另一端使用较高反射膜，目的是把更多光从指定端面输出，同时控制不需要的腔内反馈。

端面处理通常包括清洁、镀膜和检验。镀膜要在目标波长附近给出可控反射率，同时承受光功率密度、热循环和环境应力。端面缺陷、污染、氧化或局部吸收会形成热点；热点进一步增加吸收并推高温度，严重时会诱发 COD（catastrophic optical damage，灾变性光学损伤）。

| 端面相关项 | 影响 |
|---|---|
| 解理质量 | 决定腔面平整度、散射损耗和局部缺陷风险。 |
| AR / HR coating | 控制反射率、输出耦合、腔内功率分布和外部反馈敏感性。 |
| 端面吸收 | 把一部分光功率转化为局部热，影响高功率 CW 工作余量。 |
| bar-level handling | 影响端面污染、机械损伤和测试稳定性。 |
| facet reliability | 影响老化、COD 风险和长期输出功率保持率。 |

## 6. 芯片测试与筛选

裸激光器芯片进入后续组装之前，需要证明它具备足够稳定的器件级输出能力。测试会覆盖电学、光学、光谱、噪声、温度和早期失效风险。对用于高密度光互连的激光源来说，测试目标要覆盖发光之外的长期可用性：这颗芯片能否在目标电流、目标温度和目标波长窗口内持续提供可用连续光。

最基础的测试是 L-I-V 曲线。`L` 是 optical output power，`I` 是 drive current，`V` 是 diode voltage。曲线中的阈值电流、斜率效率、工作电压和串联电阻，可以同时反映有源区增益、内部损耗、电流注入效率和发热风险。高温下阈值上升或斜率效率下降过快，说明器件留给系统的热余量有限。

光谱测试用于确认 DFB 光栅和腔体是否真正选中了目标模式。中心波长决定能否落在 O-band、C-band 或 CWDM / WDM 通道附近；SMSR 描述主模相对边模的强度差；线宽和 RIN 则分别反映相位噪声和强度噪声。对连续光源来说，稳定单模、低 RIN 和可预测的温度漂移，往往比单点输出功率更能说明器件可用性。

| 环节 | 作用 |
|---|---|
| L-I-V test | 测量阈值电流、斜率效率、工作电压和输出功率。 |
| spectrum / SMSR | 判断中心波长、边模抑制和单模稳定性。 |
| RIN / linewidth | 评估噪声、频谱纯度和链路可用性。 |
| temperature sweep | 观察波长漂移、阈值漂移、功率漂移和高温工作余量。 |
| aging / burn-in | 暴露早期失效风险，观察功率衰减、阈值漂移和端面退化。 |

通过筛选的芯片仍需要基础封装、耦合、热控制和监控，才能成为可交付光源单元。更完整的 wafer-level test、known-good-die 和 package-level test 见 [Manufacturing and test](../manufacturing-test/)。

## 器件指标如何传给封装页面

光源器件把一组已测得的能力交给封装和系统设计：可用出射功率、目标波长、温度漂移、反馈容忍度、RIN、SMSR、工作电流、工作电压、热负载和寿命风险。光源放在光引擎内部、封装边缘或前面板外置模块中，属于封装与系统布局问题。对应内容见 [Package integration](../packaging/)。

激光源与封装页面之间有三条器件接口关系：

- 热接口：阈值、斜率效率、PCE 和波长温度系数决定封装需要移走多少热、允许多少结温变化。
- 光接口：输出功率、模式尺寸、发散角和反馈容忍度决定耦合结构和返回光控制的难度。
- 筛选接口：片内一致性、早期老化和失效分布决定 known-good-die、burn-in 和后续组装良率。

## 器件评价指标

| 指标类别 | 指标 | 说明 |
|---|---|---|
| 光学输出 | output power / wavelength / linewidth | 决定可用光功率、通信窗口和频谱稳定性。 |
| 噪声与单模性 | RIN / SMSR | 影响链路噪声、边模抑制和波长纯度。 |
| 电热效率 | threshold current / slope efficiency / wall-plug efficiency / PCE | 决定系统功耗和散热压力。 |
| 温度稳定性 | wavelength drift / output power drift | 决定温度变化下的波长和功率稳定性。 |
| 可靠性 | lifetime / burn-in / aging / COD | 决定长期运行能力和早期失效筛选方式。 |
| 器件接口 | feedback tolerance / coupling-ready output / monitor signal | 决定后续封装、耦合和监控能否稳定使用这颗光源。 |

## 总结：一颗激光源从哪里来

一颗 InP / III-V 激光源可以从四个层级理解：材料平台决定发光能力，外延和有源区决定光增益，波导和 DFB 结构决定模式与波长，器件输出指标决定它能否进入后续封装和光引擎。
