## 0. 什么是 InP / III-V 半导体激光源

AI 数据中心光互连链路中，光引擎需要一束稳定、低噪声的连续波光作为信号载波。这束光的起点是半导体激光器——将注入电流转化为相干光输出的 III-V 族光电子器件。本页以 InP 平台边发射 DFB 激光器为主线，覆盖从材料、外延、波导、光栅到芯片测试的完整制造路径，并最终连接到系统级应用约束。

DFB（distributed feedback，分布式反馈）利用沿光传播方向分布的周期性光栅提供波长选择和光反馈；CW（continuous-wave，连续波）指激光器以持续输出方式工作。CW-DFB 激光器即采用 DFB 结构、作为稳定连续波光源运行的器件，是硅光引擎、CPO（co-packaged optics，共封装光学）和高密度可插拔光模块的主流供光方案。

**在系统中的位置：**

```text
激光源 → 连续波光 → 耦合器 → PIC → 调制 → 光纤
```

从系统视角看，激光器是光链路的第一环。它的每一项输出参数都会沿链路向下传递，约束后续器件的设计选择：

| 激光器参数 | 下游受影响的决策 |
|-----------|-----------------|
| 中心波长 | WDM 通道匹配、微环 / AWG 设计目标 |
| 输出功率 | 链路功率预算、PIC 分光比、波导选材 |
| RIN（相对强度噪声） | 接收端信噪比、调制消光比需求 |
| SMSR（边模抑制比） | WDM 邻道串扰 |
| 线宽 | 相干通信可用调制格式 |
| 温漂系数 | 热调谐范围、TEC 需求 |
| 长期老化漂移 | 链路寿命周期功率预算余量 |
| 偏振 / PER | 耦合器选型、偏振分集方案 |
| 模场尺寸与发散角 | 边缘耦合 vs 光栅耦合 |
| 反馈容忍度 | 是否需要光隔离器 |

## 2. 工作原理：电流如何变成稳定光

激光源的工作原理由三层机制组成。

第一，电流注入与载流子复合：电流将电子和空穴注入<TermNote label="有源区" note="半导体激光器中承担电光能量转换的增益区。电流注入后，电子和空穴在这里复合并通过受激辐射放大光；通信激光器常用量子阱或多量子阱来增强载流子限制和材料增益。" />，材料体系决定这些载流子能否高效复合发光。

第二，从自发辐射到受激辐射：自发辐射先产生少量进入波导和腔体的光子；其中只有满足波导限制、腔体反馈或 DFB 光栅选择条件的光场，才能在受激辐射中被持续放大。

第三，光场约束与筛选：波导、腔体和 DFB 光栅将这些光场约束、反馈并筛选成稳定输出。

<ol className="stage-list">
  <li>
    <strong>电能转化为有序光输出</strong>
    <span>正向偏置下，电子和空穴通过 p-n / p-i-n 异质结构注入有源区。量子阱将有源区压缩到纳米级薄层，将载流子集中在一个窄区域内，提高复合概率和材料增益。注入的电子和空穴在量子阱中复合并释放光子——这是将电流转化为光的第一步。</span>
  </li>
  <li>
    <strong>InP 相关 III-V 材料覆盖通信波段</strong>
    <span>1310 nm / 1550 nm 对应的光子能量约为 0.95 eV / 0.80 eV，因此有源区需要提供接近这些能量的跃迁通道。InGaAsP、InAlGaAs 等直接带隙 III-V 合金可以通过成分设计和量子阱厚度将带隙调节至通信波段，同时保持晶格常数接近 InP 衬底，在 InP 上实现低缺陷外延生长。</span>
  </li>
  <li>
    <strong>从发光到激光：增益超过损耗</strong>
    <span>低电流时，有源区以自发辐射为主，光子方向、相位和波长分散，输出微弱。随着电流升高，注入的电子和空穴增多，受激辐射逐渐成为主导：已有光子诱导新的电子-空穴复合，产生同频率、同相位、同方向的新光子。当光在一轮传播中获得的增益足以补偿吸收、散射和输出耦合等总损耗时，器件达到激光阈值。超过阈值后，输出功率快速上升，激光器进入稳定工作状态。</span>
  </li>
  <li>
    <strong>波导、腔体和 DFB 光栅共同选择可放大的光</strong>
    <span>光在芯片中以光学模式沿波导传播。垂直方向，包层、SCH 和有源区通过折射率差将光场限制在有源区附近；横向，ridge waveguide 或 buried heterostructure 控制模式尺寸，并使光场与电流注入区域重叠；纵向，DFB 光栅通过周期性折射率扰动提供分布式反馈。光栅周期和波导有效折射率共同决定 Bragg 波长，接近该波长的前向光和后向光被强烈耦合，沿整个腔长获得反馈。光栅相移设计进一步提高目标模式的阈值优势，使输出接近稳定单一波长。</span>
  </li>
  <li>
    <strong>真实芯片还需经受热、噪声、反射和一致性挑战</strong>
    <span>高功率 CW 工作会增加结温，使带隙、折射率、增益峰和腔模位置随温度移动。载流子和光子数量的涨落表现为强度噪声（RIN），腔内自发辐射噪声和线宽增强效应影响线宽。外部反射光返回腔体时可能扰动单模输出。材料缺陷、欧姆接触质量、散热路径、片内一致性和老化筛选共同决定激光器能否长期维持稳定的输出功率和光谱质量。</span>
  </li>
</ol>

## 制造路径：从 InP 衬底到激光器芯片

器件制造可以分为六个阶段：每一阶段既包含工艺步骤，也对应形成关键激光器结构。主线止于激光器芯片和器件级输出接口，系统级封装相关内容详见封装页面。

<ol className="stage-list">
  <li>
    <strong>InP 单晶生长与衬底制备</strong>
    <span>从高纯 In / P 原料制备多晶 InP，再生长单晶晶锭，并经过切片、研磨、抛光和 CMP，得到 epi-ready InP 衬底。</span>
  </li>
  <li>
    <strong>外延生长</strong>
    <span>在 InP 衬底上生长多层 III-V 半导体结构，形成 cladding、SCH、QW / MQW 有源区和接触层。</span>
  </li>
  <li>
    <strong>波导、DFB 光栅与工艺容差</strong>
    <span>通过光刻、刻蚀、再生长或相关工艺形成 DFB 光栅、ridge waveguide 或 buried heterostructure，用来限制光场并提供反馈。</span>
  </li>
  <li>
    <strong>电流注入、绝缘与电流约束</strong>
    <span>形成 p/n contact、金属电极、钝化层和电流约束结构，使载流子能够被控制地注入有源区。</span>
  </li>
  <li>
    <strong>解理、腔长与端面工程</strong>
    <span>将晶圆加工成 laser bar / laser chip，控制腔长、端面反射、输出耦合和端面可靠性。</span>
  </li>
  <li>
    <strong>芯片测试、分档与可靠性筛选</strong>
    <span>测试阈值电流、输出功率、光谱、SMSR、RIN、温度和老化表现，并将结果用于分档与良率反馈。</span>
  </li>
</ol>

这条主线将 TOSA、ELSFP、光引擎输入和 PIC 接入归于封装与系统集成层面，只保留它们对激光源输出功率、稳定性和可靠性的反向要求。

## 3. 衬底

面向 1310 nm / 1550 nm 通信波段的半导体激光器，常建立在 InP 相关 <TermNote label="III-V 材料体系" note="由元素周期表 III 族和 V 族元素组成的化合物半导体体系，例如 InP、GaAs、InGaAsP、InAlGaAs。它们常用于高效发光、探测和高速光电子器件。" /> 上。这个选择来自目标波长、<TermNote label="有源区" note="激光器中真正产生光增益的区域。电子和空穴在这里复合，释放光子；通信激光器通常将有源区设计成量子阱或多量子阱。" />所用材料和<TermNote label="外延晶格" note="外延是在晶体衬底表面继续生长晶体薄层的工艺。新生长的材料会沿着衬底的晶格排列，因此外延材料与衬底的晶格匹配会影响缺陷和应变。" />之间的共同约束：有源区需要在通信波段形成光增益，外延层需要在低缺陷晶格模板上连续生长，InP 衬底则提供了承载这些 III-V 结构的起始平台。

### 3.1 做什么：为什么需要 InP 衬底

激光器最基本的物理原理是电子能量转换。在半导体中，电子可以被注入或激发到较高能量状态；当电子回到较低能量状态并与空穴复合时，会释放能量。若这部分能量以光子的形式释放，就产生光；若在受激辐射过程中产生的新光子与已有光子具有相同频率、相位和传播方向，光就可以被放大，形成激光。

通信波段激光器的材料选择，首先受目标光子能量约束。光子能量和波长满足近似关系 `E(eV) = 1.24 / λ(μm)`，所以 1310 nm 约对应 0.95 eV，1550 nm 约对应 0.80 eV。激光器的有源区需要具有接近这些能量的 <TermNote label="带隙" note="半导体中价带顶和导带底之间的能量差。电子从导带回到价带并与空穴复合时，释放出的能量大致由带隙决定。" />，并且要能在电注入后高效产生辐射复合和光增益。

对发光器件来说，还需考虑这种复合是否能够高效地以光子形式释放能量。<TermNote label="直接带隙" note="导带底和价带顶位于相近的晶体动量状态，电子和空穴复合时更容易直接放出光子，因此更适合高效发光。" />材料更适合作为激光器有源区；<TermNote label="间接带隙" note="导带底和价带顶不在相同动量位置，复合时通常还需要声子参与，能量更容易变成晶格振动和热，发光效率较低。" />材料则更难作为高效原生光源。InP 相关 III-V 材料体系的意义在于，它提供了适合高效发光的直接带隙材料，并且可以通过调整 InGaAsP / InAlGaAs 等合金的元素比例，将有源区带隙调节至 1310 nm / 1550 nm 通信波段所需的光子能量附近。

纯 InP 的室温带隙约 1.34 eV，对应比通信波段更短的波长。通信波段激光器的主要光增益通常来自 InGaAsP 或 InAlGaAs <TermNote label="量子阱" note="夹在较大带隙材料之间的极薄半导体层，可以将电子和空穴限制在很窄的区域内，提高复合概率和光增益效率。" /> / 多量子阱。选择 InP 衬底的核心原因在于平台匹配：这些有源区材料可以通过成分设计将带隙调节至通信波段，同时让晶格常数接近 InP，从而在 InP 上形成低缺陷外延结构。

这一材料选择可以从三个层面理解：

- **带隙层面**：有源区材料的带隙决定可产生的光子能量，因而决定目标波长附近能否形成增益峰。
- **晶格层面**：有源区、限制层和包层需要沿衬底晶格连续生长，晶格常数接近时，应变和位错更容易控制。
- **器件层面**：InP 平台可以承载 InP cladding、SCH、InGaAsP / InAlGaAs QW、接触层、DFB 光栅和波导加工，形成成熟的通信波段激光器工艺链。

### 3.2 怎么做：晶格匹配与 InP 衬底制造

从晶体物理看，衬底提供了外延生长的原子排列周期模板。所谓 <TermNote label="晶体" note="原子或离子按照周期性规则排列形成的固体。半导体晶体中的周期结构决定了能带、晶向、缺陷和外延生长方式。" />，是原子在三维空间里按照固定周期重复排列形成的固体。描述这个重复单元尺寸的关键参数之一，就是 <TermNote label="晶格常数" note="晶体周期结构中的基本长度尺度，可以理解为原子排列的节距。外延层和衬底晶格常数接近时，应变和位错更容易控制。" />。InP 采用闪锌矿（zincblende）晶体结构，晶格常数约为 5.87 A。

外延生长时，新沉积的 III-V 材料会沿着衬底表面的晶格继续排列；如果外延层的本征晶格常数接近 InP，就能以较低应变长成连续薄层。原子沿已有晶体模板继续规则排列：模板越平整、周期越匹配，后续薄层越容易保持同样的晶体秩序。

InGaAsP 和 InAlGaAs 这类多元 III-V 合金的价值在于带隙和晶格常数可以协同设计：一方面通过成分比例调节带隙，使电子空穴复合释放的光子能量对应 1310 nm / 1550 nm 附近的波长；另一方面让材料晶格常数接近 InP，使有源区、限制层和包层能够沿 InP 晶格模板生长。对激光器来说，带隙决定光增益能否位于目标波段，晶格匹配决定这个有源区能否以低缺陷方式外延生长。

如果外延层和衬底之间存在较大 <TermNote label="晶格失配" note="外延层本征晶格常数与衬底不同，生长时会产生应变。常规量子阱 / DFB 激光器通常要控制失配以避免位错；受控应变也可用于能带工程和特定量子结构设计。" />，薄层会先被拉伸或压缩。应变超过材料能够承受的范围后，晶体会以 misfit dislocation、threading dislocation 等缺陷释放应变。这些缺陷会穿过有源区或靠近有源区，成为非辐射复合中心、漏电通道和长期失效起点。载流子本应在 QW / MQW 有源区里复合并贡献光增益，缺陷会将一部分能量转化为热和损耗，表现为阈值电流升高、斜率效率下降、暗电流增大和可靠性变差。

因此，InP 衬底这一环节所决定的变量很多：晶格模板、晶向、缺陷底噪、表面粗糙度、污染水平、翘曲和厚度一致性。后续外延、光刻、解理和测试都会将这些初始条件进一步放大。


InP 衬底制造的目标，是将元素级 In / P 原料转化为化学计量、晶向、缺陷密度和表面质量都可控的 InP 晶圆。

第一步是高压合成多晶 InP。磷属于高蒸气压 V 族元素，高温下容易挥发；合成时需要高压环境、受控温场和能够承受反应条件的反应器。典型设备包括高压石英玻璃反应器、分区加热系统、惰性气体 / 压力控制、热电偶和装载铟的坩埚。工艺上，磷先在较低温区升华成气相，再输运到液态铟附近并反应形成 InP。反应完成后，通过定向凝固和蚀刻减少非化学计量成分、夹杂和残余污染，为后续单晶生长准备原料。

第二步是单晶生长。VGF（Vertical Gradient Freeze）等工艺将多晶 InP 重新熔化，并在受控温度梯度下推进固液界面，使晶体沿种晶方向长成单晶晶锭。典型 VGF 系统会将预合成 InP 放入 <TermNote label="pBN 坩埚" note="pyrolytic boron nitride，热解氮化硼坩埚，常用于高温化合物半导体生长，具有较高化学稳定性并可降低对熔体的污染。" />，再放入多区控温炉中；炉体通过不同加热区形成熔点上下的近恒温区和较小的轴向温度梯度。晶体通常从定向种晶开始，固液界面向上推进。关键控制量包括温度梯度、界面形状、生长速率、掺杂均匀性和热应力。温度场越不稳定，越容易引入位错、应力和晶向偏差；这些问题会传递到后续外延层。

第三步是晶圆加工。单晶晶锭需要先定向，确定晶向和切片角度；随后通过内圆切割或线切割将晶锭切成晶片。线切割会用携带磨料的线穿过晶体，效率高，但也会引入切割损伤和表面应力。切片后还要经过边缘倒角、表面研磨、刻蚀、抛光和 CMP，将晶片加工成厚度、平坦度、粗糙度都满足外延要求的 wafer。

第四步是清洗、检测和分级。清洗线需要去除颗粒、有机 / 无机薄膜和残余抛光液，并重新控制表面氧化层状态。检测环节会用高分辨率显微镜检查可见缺陷，用腐蚀坑密度 <TermNote label="EPD" note="etch pit density，腐蚀坑密度。对 InP 可用含 HBr 的腐蚀液显露位错穿出表面的位置，再通过扫描计数来评估晶体位错密度。" /> 评估位错，用 Hall / van der Pauw 测量电学参数，也可用非接触电阻率映射观察晶圆内均匀性。达到外延要求后，这片晶圆才可以称为 epi-ready InP 衬底。

从设备角度看，InP 衬底制造可以视为一条从高压化学合成到精密表面处理的工具链：

| 环节 | 典型器材或工具 | 作用 |
|---|---|---|
| 高压合成 | 高压石英反应器、分区加热系统、气压控制、热电偶、坩埚 | 让高纯 In / P 在可控压力和温度下形成多晶 InP，并控制磷挥发。 |
| VGF 长晶 | pBN 坩埚、多区控温炉、种晶、温度梯度控制系统 | 控制固液界面、晶向和热应力，将多晶料转化为低位错单晶晶锭。 |
| 定向与切片 | X-ray 定向、内圆切割或线切割设备 | 确定晶向和切片角度，将晶锭加工成晶片。 |
| 研磨与 CMP | 研磨盘、抛光垫、化学机械抛光液、双面 / 单面抛光工具 | 去除切割损伤，控制厚度、平坦度、粗糙度和表面损伤层。 |
| 清洗与检测 | 超声 / 化学清洗、显微镜、EPD 扫描、Hall / van der Pauw、非接触电阻率映射 | 控制颗粒、污染、位错、电学均匀性和 epi-ready 表面状态。 |

| 工艺关注点 | 对激光源的影响 |
|---|---|
| 原料纯度 | 金属杂质和残余污染会缩短载流子寿命，增加暗电流和早期失效概率。 |
| 化学计量 | In 与 P 的比例和 V 族元素挥发需要受控，避免形成富 In / 富 P 缺陷和夹杂。 |
| 单晶质量 | 位错密度、夹杂和微裂纹会影响外延连续性，并可能成为器件失效起点。 |
| 几何一致性 | 晶圆厚度、TTV、bow 和 warp 会影响外延均匀性、光刻焦深、解理方向和后续测试稳定性。 |

### 3.4 量产壁垒

InP 衬底制造的壁垒，在于如何长期、批量地提供低缺陷、低污染、几何一致且可直接进入外延的 epi-ready 晶圆。磷在高温下具有较高蒸气压，InP 合成需要高压环境、稳定温场和化学计量控制；VGF 长晶又要控制固液界面、温度梯度、热应力和掺杂均匀性，才能将位错、夹杂和晶向偏差控制在（或降低至）外延可接受的范围内。

进入晶圆加工后，竞争力继续体现在定向、切片、研磨、抛光、CMP、清洗和检测的组合能力上。InP 晶体较脆，切片和倒角容易引入崩边、表面损伤和残余应力；表面粗糙度、颗粒、金属污染、TTV、bow、warp 和 EPD 分布又会影响后续外延均匀性、光刻稳定性、解理质量和器件可靠性。因此，成熟衬底厂商的核心能力通常是材料纯度、长晶炉温场经验、低缺陷晶锭良率、epi-ready 表面处理、片内 / 片间一致性、计量分级体系，以及与下游外延和激光器客户共同对齐规格所积累的长期数据。

## 4. 外延

外延生长将上一节得到的 epi-ready InP 衬底转变为激光器的功能材料堆叠。衬底提供晶体模板，外延工艺在这个模板上逐层生长 III-V 薄层：有些层用于导电，有些层用于限制光场，有些层则将电子和空穴约束在极薄区域内并产生光增益。对半导体激光器来说，外延质量基本决定了后续器件加工能够达到的性能上限。

### 4.1 做什么

外延的本质是"沿着已有晶体模板继续排列"。InP 衬底表面已经具有周期性的原子排列；当 In、Ga、Al、As、P 等原子到达加热后的表面时，它们会在合适位置吸附、迁移并并入晶格，形成新的单晶薄层。每一层的材料成分、厚度、掺杂和界面质量，由反应气体、温度、压力和生长时间决定。

这一阶段要同时解决三个问题：

- **材料成分**：InGaAsP / InAlGaAs 的组分决定带隙，进而决定有源区能否在 1310 nm / 1550 nm 附近提供光增益。
- **晶体连续性**：外延层需要沿 InP 晶格模板以低缺陷密度生长，避免位错、界面粗糙度和应变失控。
- **空间分工**：激光器需要将 <TermNote label="载流子" note="在半导体中能够携带电流的粒子，主要指电子和空穴。激光器里，电子从 n 侧注入、空穴从 p 侧注入，并在有源区复合产生光增益。" /> 限制在有源区，将光场限制在 <TermNote label="波导" note="引导光在芯片内沿特定路径传播的结构。激光器里，波导通过折射率差和几何边界将光场限制在有源区附近，提高光场与增益区的重叠。" /> 附近，同时还要提供低阻电流注入路径。

### 4.2 怎么做：MOCVD 与 MBE

量产 III-V 光电子器件常使用 MOCVD（Metal-Organic Chemical Vapor Deposition，金属有机化学气相沉积）。MOCVD 将含 III 族元素的金属有机前驱体和含 V 族元素的气体送入反应腔，在加热晶圆表面分解并形成单晶外延层。反应器通常包含气体混合与输运系统、金属有机前驱体源瓶、氢气或氮气载气、加热基座、多区温控、压力控制、尾气处理和晶圆旋转结构。前驱体气体流过加热的 InP 晶圆表面后分解，原子并入晶格；调节前驱体比例可以改变 In/Ga/Al/As/P 的相对含量，切换掺杂气体可以形成 p 型或 n 型层，控制生长时间则决定薄层厚度。

MBE（Molecular Beam Epitaxy，分子束外延）则提供了另一条技术路径：在超高真空中将元素或分子束入射到晶圆表面，并通过独立源炉、快门和束流控制 In、Ga、Al、As、P 等材料供给。MBE 的优势是界面切换清晰、厚度控制精细；MOCVD 的优势是产能、均匀性和产业成熟度更适合大规模化合物半导体制造。

两种设备路线的差异可以概括为：MOCVD 主要通过气体流量、V/III 比、温度、压力和旋转均匀性来控制外延；MBE 主要通过超高真空、源炉温度、快门切换、束流监测和 RHEED 原位观察来控制外延。

```text
epi-ready InP substrate — 外延起始表面
→ MOCVD / MBE epitaxy — 外延生长多层半导体结构
→ cladding / SCH / MQW / contact layers — 形成包层、限制层、有源区和接触层
```

### 4.2 怎么做：外延层分工

一个边发射 InP 激光器的外延结构，可以按从衬底往上的垂直堆叠来理解。实际器件会因 DFB、EML、buried heterostructure 或 ridge 结构而变化，但核心功能层通常围绕电流注入、载流子限制、光场限制和光增益展开：

| 层级 | 常见材料或结构 | 作用 |
|---|---|---|
| InP 衬底（InP substrate） | n-InP 衬底 | 作为外延生长的晶格模板、机械支撑和下侧电流路径。 |
| 下包层（lower cladding） | n-InP 或相关 InP 层 | 通过较低折射率从下侧限制光场，并承担 n 侧电流注入。 |
| 下 SCH / 波导限制层 | InGaAsP / InAlGaAs 限制层 | 将光场引向有源区附近，并与量子阱形成合适的光场重叠。 |
| 量子阱有源区（QW / MQW active region） | InGaAsP 或 InAlGaAs 量子阱 / 多量子阱 | 电子和空穴在该区域复合并产生光增益，是决定波长、阈值和效率的核心区域。 |
| 上 SCH / 波导限制层 | InGaAsP / InAlGaAs 限制层 | 从上侧配合下 SCH 限制光场，并帮助载流子和光场在有源区附近重叠。 |
| 上包层（upper cladding） | p-InP 或相关 InP 层 | 通过较低折射率从上侧限制光场，并提供 p 侧电流注入路径。 |
| 接触层（contact layer） | 高掺杂 InGaAs / InGaAsP 等 | 降低金属接触电阻，为后续电极加工提供界面。 |

这张表按垂直堆叠顺序排列，不表示各层重要性相同。QW / MQW 有源区是产生光增益的核心层；上下 <TermNote label="SCH" note="Separate Confinement Heterostructure，分离限制异质结构。它将极薄量子阱放在较宽的波导/限制层中，使载流子主要留在量子阱，光场则分布在更宽的波导区域。" /> / 波导限制层决定光场与有源区的重叠；包层、接触层和掺杂结构则负责将光和电流约束到正确位置。这些结构对波长、阈值、效率、损耗和可靠性的影响，会在外延设计变量中体现出来。

<TermNote label="量子阱" note="quantum well，夹在较大带隙材料之间的极薄低带隙半导体层。电子和空穴在厚度方向被限制，复合概率提高，光增益和阈值电流更容易被工程化控制。" />厚度通常只有纳米量级。多个量子阱叠在一起形成 MQW，可以在保持载流子限制的同时增加总增益。量子阱之间的 barrier 层负责隔开各个阱，并帮助调节应变、载流子分布和光学模式重叠。

Optical confinement layer 指外延材料堆叠里的垂直光场限制区域：光场主要由包层、SCH / waveguide core 和有源区共同限定。ridge waveguide、buried heterostructure 等横向波导结构通常还需要后续光刻、刻蚀、再生长或钝化工艺形成，属于器件加工环节。

### 4.3 对器件的影响

外延结构将"材料具备发光能力"转化为"器件可稳定输出"。其影响可以分成几类：

| 外延设计项 | 对激光源的影响 |
|---|---|
| 有源区带隙 | 决定光增益峰值落在哪个波段，并影响温度漂移。 |
| 量子阱厚度与数量 | 影响阈值电流、微分增益、饱和输出功率和高温效率。 |
| SCH 与包层折射率 | 决定光场限制、模式尺寸、内部损耗和与后续波导结构的匹配。 |
| 掺杂分布 | 影响串联电阻、电流扩展、自由载流子吸收和局部发热。 |
| 应变与晶格匹配 | 影响位错形成、可靠性和长期老化。 |
| 界面粗糙度与厚度误差 | 造成散射损耗、增益不均匀和波长偏移。 |

通信波段常用 InGaAsP / InAlGaAs 等有源区材料覆盖 1310 nm 和 1550 nm 附近窗口。InAlGaAs/InP 体系具有较大的导带偏移，有助于在高温下限制电子泄漏；InGaAsP/InP 工艺成熟，常用于通信波段 DFB 和 EML 结构。实际选择通常取决于目标波长、输出功率、温度范围、阈值、效率、可靠性和制造成熟度。

### 4.2 怎么做：工艺控制

外延看似只是生长材料，实际控制对象是一组相互耦合的物理变量：

| 控制对象 | 典型控制方式 | 影响 |
|---|---|---|
| 成分比例 | 调节 III 族 / V 族前驱体流量、V/III 比和生长温度。 | 决定带隙、晶格常数、应变和折射率。 |
| 厚度 | 控制生长速率和生长时间，也可结合原位监测校准。 | 决定量子阱能级、波导模式和 DFB 目标波长的基础偏差。 |
| 掺杂 | 引入 Zn、Si、S 等掺杂源并控制浓度剖面。 | 决定 p/n 区导电性、接触电阻、吸收损耗和热负载。 |
| 界面质量 | 控制气体切换、温度稳定、反应腔记忆效应和表面洁净度。 | 决定非辐射复合、散射损耗和器件一致性。 |
| 晶圆均匀性 | 优化气流、温场、压力、晶圆旋转和装片方式。 | 决定同一片晶圆上阈值、波长和输出功率的一致性。 |

### 4.4 量产壁垒

外延层的公开知识和量产配方属于不同层级。公开资料通常能够说明 InP 激光器为什么采用 InGaAsP / InAlGaAs 有源区、为什么需要 QW / MQW、SCH 和包层，也能看到一些论文或讲义给出示例结构、波导设计、量子阱数量、腔长、内部损耗和量子效率。这些内容构成行业共识，足以理解器件原理和主要设计变量。

进入制造环节后，同样的层名会变成更细的工艺问题：每层的实际厚度、成分比例、应变量、p/n 掺杂浓度和掺杂剖面，需要与具体 MOCVD / MBE 设备、反应腔历史、前驱体纯度、气体切换、温度场、晶圆旋转、装片方式和在线 / 离线计量相互匹配。公开论文中的某个层厚或掺杂值可以作为设计例子，不能直接等同于另一条产线的可复制配方。

InP 激光器厂商的核心壁垒通常由两部分组成：理解器件应该有哪些功能层，以及将目标波长、阈值电流、输出功率、RIN、SMSR、温度漂移和寿命要求，稳定地转化为可量产的外延窗口和筛选规则。后者包括生长速率校准、V/III 比控制、掺杂记忆效应、界面切换、片内均匀性、再生长或钝化兼容性，以及外延数据与后续光刻、解理、测试、老化结果之间的反馈闭环。能长期保持良率和可靠性的参数窗口，往往比单张外延结构示意图更难复制。

外延已将激光器的波长、阈值、效率、温度敏感性和寿命风险固化在材料堆叠结构中。当激光源用于 CPO 或其他高密度光互连场景时，高功率连续工作和较高温度稳定性会使有源区设计、限制层设计、掺杂和缺陷控制变得更关键。

## 5. 波导与光栅

外延层已经确定了垂直方向的材料结构：低折射率包层、高折射率限制层和有源区共同决定光场在厚度方向上的分布。器件加工继续完成另外两个方向的控制：横向上将光场和电流约束在窄条区域内，纵向上用腔体和 DFB 光栅选择稳定中心波长和纵模。这个环节的因果链可以概括为：几何结构决定光场与电流分布，光场分布决定 DFB 反馈强度，反馈和损耗共同决定阈值、SMSR、效率和可靠性。

### 5.1 做什么

InP 激光器里的波导要让光场、载流子和增益区在空间上重合。InP 既作为衬底和包层材料，也是 III-V 族半导体光子集成平台的基础；通信波段的主要光增益通常来自 InGaAsP / InAlGaAs 量子阱或多量子阱有源区。波导结构的任务，是将传播光场限制在这些有源层附近，使注入载流子产生的增益能被有效利用。

光在半导体芯片中传播时，会优先被约束在折射率较高且满足边界条件的区域。这个区域称为波导的芯层（core）；周围折射率较低的材料称为包层（cladding）。真实光场会以 <TermNote label="光学模式" note="optical mode，满足波导边界条件的一种稳定电磁场分布。它决定光场中心、扩展范围、偏振和与有源区的重叠。" /> 的形式分布在 core、cladding 和有源区附近，并向几何 core 外侧延伸。折射率分布提供 index guiding，电流注入区域带来的增益分布也会影响模式位置；在实际器件中，这两种作用常常同时存在。

因此，波导设计最终控制的是几个与性能直接相连的量：<TermNote label="有效折射率" note="effective index，常记作 n_eff。它表示一个光学模式在复杂波导结构中传播时看到的等效折射率，会随材料、几何尺寸、温度和载流子分布变化。" />、<TermNote label="光学限制因子" note="optical confinement factor，常记作 Gamma。它表示光学模式有多少比例与有源区或某个材料区域重叠，影响增益利用率、阈值和耦合强度。" />、横向单模条件，以及光场与量子阱有源区的重叠程度。

边发射 InP 激光器通常需要同时控制三种约束：

| 约束方向 | 主要结构 | 作用 |
|---|---|---|
| 垂直方向 | 外延包层、SCH、量子阱和波导芯层 | 将光场约束在有源区附近，提高光场与增益区的重叠。 |
| 横向方向 | ridge waveguide、buried heterostructure、stripe aperture | 限制光场和电流的横向扩展，降低横向多模和电流扩散。 |
| 纵向方向 | 端面、DBR / DFB 光栅、相位移区域 | 决定反馈、腔长、模式选择和目标波长。 |

这对应两个不同层级：SCH / optical confinement layer 主要来自外延材料堆叠，解决垂直方向的光场限制；ridge waveguide 和 buried heterostructure 属于器件加工形成的横向结构，解决光场与电流在芯片宽度方向的分布。

### 5.2 怎么做：Ridge 与 BH 加工

Ridge waveguide 的基本做法，是先用光刻定义一条窄脊，再通过刻蚀移除脊两侧的上层材料。脊区保留较高的等效折射率，两侧经刻蚀减薄或被介质覆盖后，横向等效折射率下降，光学模式便更容易被局限在脊下方的有源区附近。Ridge 还会定义电流注入区域：芯片表面通常覆盖绝缘 / 钝化层，只在脊顶部开出金属接触窗口；因此电流主要从脊正下方穿过外延层，进入与光学模式重叠的有源区，两侧的大面积电流扩散会被抑制。

Buried heterostructure（BH）在激光器语境里通常指将有源条带埋入周围材料或电流阻挡层中的结构。器件先刻出包含有源区的 mesa 或窄条，再通过再生长将两侧埋入 InP、半绝缘 InP 或其他电流阻挡结构中。被埋入的结构可以同时提供横向折射率约束和电流阻挡，适合低阈值、单横模和良好热稳定性的边发射激光器。

BH 的难点来自"先刻蚀、再包埋"的工艺顺序：刻蚀会暴露有源条带侧壁，后续再生长必须在这些侧壁上形成干净、低缺陷的界面；两侧的 InP 或电流阻挡层也要具有合适的掺杂和电阻，才能减少侧向漏电。界面污染、侧壁损伤或阻挡层设计失控，最终表现为阈值升高、效率下降、漏电增加和寿命缩短。

| 结构 | 典型工艺 | 优点 | 需要控制的风险 |
|---|---|---|---|
| ridge waveguide | 光刻、硬掩膜、干法或湿法刻蚀、介质钝化、金属开窗 | 工艺相对直接，便于定义横向模式和电流孔径。 | 侧壁粗糙度、刻蚀深度、脊宽和介质应力会影响损耗、阈值和可靠性。 |
| buried heterostructure / BH | 台面刻蚀、表面清洁、InP 或阻挡层再生长、平坦化和接触加工 | 横向光场与电流约束更强，热和模式控制更好。 | 再生长界面缺陷、电流泄漏、阻挡层设计和应力控制更难。 |
| laterally coupled grating | 在有源区两侧或波导邻近区域制作周期结构 | 光栅可避开直接破坏有源区，便于调节耦合强度。 | 光栅与模式重叠较弱时，需要更精确的几何和耦合设计。 |

### 5.2 怎么做：DFB 光栅

DFB 是 distributed feedback，指沿光传播方向分布的反馈结构。它依靠一系列周期性折射率扰动提供反馈。每一个光栅周期都会产生很弱的反射；当这些微反射在某个波长上相位叠加，前向传播的光便会被有效耦合成后向传播的光，形成分布式反馈。

DFB 光栅的 Bragg wavelength（Bragg 波长）可粗略写成：

```text
lambda_B = 2 n_eff Lambda / m
```

其中 `lambda_B` 是 Bragg 波长，`n_eff` 是波导模式看到的有效折射率，`Lambda` 是光栅周期，`m` 是 Bragg 阶数。这一关系说明两点：光栅周期直接决定目标波长，有效折射率也会将材料、温度、波导尺寸和工艺误差带入波长结果。以 1550 nm 附近的 InGaAsP / InP DFB 激光器为例，若 `n_eff` 约为 3.4，一阶光栅周期大约是 0.23 μm，二阶光栅周期约为 0.46 μm；实际设计会随波导结构和工艺路线变化。

DFB 光栅更接近于沿腔长方向分布的模式选择结构：它将前向波和后向波耦合起来，使接近 Bragg 条件的纵模获得阈值优势。光栅设计还需关注 <TermNote label="耦合系数" note="grating coupling coefficient，常记作 kappa，表示光栅将前向波耦合到后向波的强度。它由折射率扰动、光栅深度、占空比、长度以及光学模式与光栅的重叠决定。" />。耦合太弱，反馈不足，阈值和边模抑制受影响；耦合太强，光栅损耗和空间烧孔等问题会变得突出。实际 DFB 设计还常加入四分之一波相移或等效相位设计，将目标模式变为最低阈值模式，从而提高单模稳定性和 SMSR。

光栅可以位于有源区附近的埋入层，也可以做成表面光栅或侧壁 / 横向耦合光栅。埋入式光栅常带来更强、更可控的模式重叠，但往往需要额外外延或再生长；表面或侧壁光栅工艺更直接，同时会将侧壁粗糙度、散射损耗和耦合强度一致性纳入容差问题。

### 5.3 对器件的影响

波导与 DFB 光栅加工将纳米级几何误差转化为光学指标的偏差。对通信激光器来说，这些误差会沿着"几何变化 -> 光场 / 反馈变化 -> 器件指标变化"的链路，传导至波长、阈值、边模抑制、线宽、RIN、损耗和长期可靠性等指标。

| 工艺变量 | 物理影响 | 器件表现 |
|---|---|---|
| 光栅周期 `Lambda` | 直接改变 Bragg 条件。 | 中心波长偏移，WDM 通道对准变差。 |
| 有效折射率 `n_eff` | 受外延厚度、材料组分、波导尺寸、温度和载流子密度影响。 | 波长随晶圆位置、温度和电流漂移，批次一致性变差。 |
| 光栅深度 / 占空比 / 位置 | 改变 `kappa`、散射损耗和光栅与光场的重叠。 | SMSR、阈值电流、斜率效率和线宽发生变化。 |
| 光栅相位 / 拼接误差 | 扰乱沿腔体的反馈相位关系。 | 单模选择变弱，边模抑制下降。 |
| 波导宽度 / 刻蚀深度 | 改变横向模式、`n_eff` 和电流孔径。 | 横向多模、模式跳变、波长偏移或局部发热风险上升。 |
| 侧壁粗糙度 / 刻蚀损伤 | 增加散射、后向反射和非辐射复合中心。 | 阈值升高，输出功率、RIN 和可靠性下降。 |
| BH 再生长界面 / 电流阻挡层 | 引入界面缺陷、污染、应力或侧向漏电。 | 漏电、非辐射复合、效率下降和早期失效概率增加。 |

实际使用中，外部反射也会反馈到 DFB 腔体。光源器件需要通过端面镀膜、反馈容忍度设计、较低 RIN 和稳定单模工作来降低这类扰动；后续封装和系统还会继续处理连接器反射、光隔离和耦合回波。

### 5.4 量产壁垒

波导和 DFB 光栅的竞争力，集中在将纳米级几何结构稳定地转化为可预测的光学指标。公开资料可以说明 ridge、BH、DFB 光栅和 Bragg 条件的基本原理；量产中的难点，是让这些结构在整片晶圆、多个批次和长期老化后仍保持一致的 `n_eff`、`kappa`、损耗、阈值和中心波长。

成熟厂商的优势通常来自几类能力：光刻和刻蚀可以长期控制脊宽、刻蚀深度、侧壁粗糙度和光栅周期；再生长和清洗可将 BH 或埋入式光栅界面的污染、缺陷和应力降至较低水平；测试数据可以将波长偏移、SMSR、阈值、斜率效率、RIN 和老化结果反向关联到工艺窗口。对通信激光器来说，真正难复制的是材料、图形、刻蚀、再生长和筛选规则之间的闭环，而不只是某一个截面形状。

## 6. 电流注入

半导体激光器是电流驱动的器件。外部驱动电路提供电流，电极和半导体接触层将该电流注入 p-n / p-i-n 异质结构；电子和空穴进入量子阱有源区后复合，才可能形成光增益。这个环节的因果链可以概括为：电极将电流注入器件，欧姆接触降低界面电压损耗，绝缘开窗决定电流注入位置，电流约束结构抑制横向扩散和旁路漏电，最终影响阈值、效率、温漂和可靠性。

### 6.1 做什么

一个边发射激光器的光沿芯片长度方向传播，电流通常沿垂直方向穿过外延层。上方 p-contact / metal 连接 p 侧接触层和 p-InP 包层，下方 n-contact 或衬底背面金属提供 n 侧回路。电流通过这些层后，电子和空穴需要进入 QW / MQW 有源区，并与上一节讨论的光学模式在空间上重合。

这个路径受三个几何关系控制：

| 几何关系 | 物理含义 | 影响 |
|---|---|---|
| 电流路径与有源区重叠 | 电流是否主要穿过 QW / MQW 区域。 | 决定阈值电流、效率和局部发热。 |
| 电流孔径与光学模式重叠 | 被注入的载流子是否覆盖主要光场区域。 | 决定增益利用率和横向模式稳定性。 |
| 电极 / 绝缘窗口与波导位置 | 金属、介质和刻蚀边界是否对准 ridge、mesa 或有源条带。 | 决定串联电阻、局部发热、漏电和可靠性。 |

### 6.2 怎么做：欧姆接触

金属直接接触半导体时，界面可能形成势垒，使电流通过界面时出现额外压降。激光器需要的是 <TermNote label="欧姆接触" note="ohmic contact，低阻、近似线性的金属-半导体接触；它让电流在工作电压下稳定进入半导体，避免界面表现出明显整流行为。" />。外延顶部通常会设计高掺杂 InGaAs 或 InGaAsP 接触层，使金属更容易将电流注入 p 侧；金属叠层沉积后，常通过退火或合金化降低接触电阻。n 侧通常通过导电衬底背面金属或 n-contact 完成电流回路。

接触电阻过高时，外部电压的较大部分会损耗在接触界面和半导体串联电阻上，转化为焦耳热而非光输出。局部发热会改变带隙、折射率和增益峰位置，使波长漂移、阈值升高、RIN 变差，并加速金属扩散、界面反应和缺陷增长。接触层、金属体系、退火窗口和界面洁净度，因此会同时影响效率和长期可靠性。

### 6.2 怎么做：绝缘与钝化

电极不能让整片暴露的半导体表面都参与导电。工艺上通常先沉积 SiN、SiO2 或类似介质层，对刻蚀后的侧壁和表面进行钝化；随后通过光刻和刻蚀在需要注入电流的位置打开窗口，再沉积金属。这个窗口决定电流的注入位置，位置和宽度需要与 ridge、mesa 或有源条带中心对齐。

介质层还兼有钝化功能。刻蚀侧壁附近容易存在悬挂键、污染和等离子损伤，这些位置会形成表面漏电通道或非辐射复合中心。介质层覆盖不完整、应力过大或开窗边缘不稳定时，电流会偏离预定注入区域，产生局部热点和早期失效风险。窗口过宽会增加横向扩散，窗口过窄则可能提高接触电阻和局部电流密度。

### 6.2 怎么做：电流约束

上一节介绍 ridge 和 BH 时，重点是它们如何限制光场；从电学角度看，同一类结构还要限制电流。理想情况是载流子主要进入光学模式覆盖的有源区，应尽可能减少旁路电流、侧向漏电和无效复合。

Ridge 结构里，电流主要通过脊顶部的窗口注入，侧壁和周围区域通过介质层隔离。窗口太宽，电流更容易向两侧扩散，在光场较弱的位置复合并产生热量；窗口太窄，接触电阻和局部电流密度会升高。Stripe contact 也可以定义窄条注入区域，但横向光场和电流扩散控制通常弱于经过刻蚀定义的 ridge 或更强约束结构。

BH / buried heterostructure 结构里，有源 mesa 或有源条带被再生长材料包围，周围材料可以提供横向折射率约束，也可以通过半绝缘 InP、反偏 p-n 结或其他 current blocking layer 减少旁路电流。BH 的优势是电流、载流子和光场更容易集中在同一条窄区域内；代价是再生长界面、阻挡层掺杂、材料扩散和界面洁净度会引发良率与可靠性问题。高温工作时，阻挡层尺寸和掺杂设计还会影响漏电电流。

| 结构 | 主要电学作用 | 需要控制的风险 |
|---|---|---|
| dielectric aperture | 通过绝缘介质开窗定义电流注入位置。 | 开窗偏移、边缘缺陷和介质应力会造成局部发热或漏电。 |
| stripe contact | 用窄条金属接触限制上表面注入范围。 | 横向扩散较明显，电流与光场重叠不稳定。 |
| ridge waveguide | 结合刻蚀几何、介质隔离和顶部窗口限制横向电流。 | 脊宽、刻蚀深度、侧壁钝化和窗口宽度会共同影响电阻、漏电和模式稳定性。 |
| buried heterostructure / current blocking layer | 通过再生长材料或阻挡层减少有源区两侧旁路电流。 | 再生长界面、阻挡层掺杂和高温漏电控制更难。 |

### 6.3 对器件的影响

电流注入工艺的误差，会沿着"工艺变化 -> 电流分布 / 电阻 / 漏电 / 发热变化 -> 器件指标变化"的链路，传导至输出表现：

| 工艺变量 | 物理影响 | 器件表现 |
|---|---|---|
| 接触电阻偏高 | 提高工作电压和焦耳热。 | PCE 下降，结温上升，阈值和波长温漂加重。 |
| 金属扩散 / 界面反应 | 污染接触层或改变金属-半导体界面。 | 串联电阻漂移，长期输出功率和可靠性下降。 |
| 绝缘开窗偏移 | 电流注入位置偏离 ridge、mesa 或光学模式中心。 | 有效增益下降，局部热点和模式不稳定风险上升。 |
| 侧壁钝化不足 | 增加表面漏电和非辐射复合。 | 阈值升高，斜率效率下降，早期失效概率增加。 |
| 电流孔径宽度失控 | 改变电流密度和横向扩散范围。 | 过宽会增加无效复合，过窄会增加电阻和局部发热。 |
| 阻挡层设计失控 | 让一部分电流绕开 QW / MQW 有源区。 | 外微分效率和 PCE 下降，高温下漏电更明显。 |

因此，在激光器截面中，电流路径、光学模式和有源区位置需要同时满足：载流子主要进入有光场的区域，金属和接触层提供低阻路径，绝缘和阻挡结构减少旁路漏电、侧壁复合和局部过热。做到这一点，外延层提供的材料增益才能真正转化为低阈值、高效率和可长期工作的激光源。

### 6.4 量产壁垒

电流注入环节的壁垒，集中在如何将电流稳定地注入预定区域。公开结构图可以说明 p-contact、介质开窗、ridge、BH 和 current blocking layer 的基本形态；难点在于让低接触电阻、低漏电、低局部发热和长期稳定性同时满足。

成熟工艺通常要长期控制几组变量：接触层掺杂、金属叠层、退火窗口和界面洁净度决定欧姆接触；介质层应力、开窗尺寸和对准精度决定电流注入位置；侧壁钝化和刻蚀损伤控制表面漏电和非辐射复合；BH 或阻挡层设计决定高温下是否出现旁路电流。能够将 I-V、L-I-V、温度扫描和老化数据映射回这些工艺变量，是电流注入环节形成技术优势的关键。

## 7. 端面工程

完成外延、光栅、波导、电极和介质层加工后，激光器仍然处在晶圆形态。边发射 InP 激光器还需要沿晶体解理面形成 laser bar，再从 bar 分割成单颗 laser chip。这个步骤同时改变器件形态和光学边界：解理面会成为出光端面，端面反射率、端面相位和端面质量会影响输出功率、SMSR、寿命和良率。

### 7.1 做什么：从晶圆到 bar

前面几节的大多数工艺发生在完整晶圆上；到了解理阶段，晶圆被分成带有多个发光条的 laser bar，两个解理面暴露出每颗边发射激光器的前后端面。bar 阶段通常便于进行端面清洁、镀膜、检查和部分测试，随后再切分成单颗 die / chip。

该流程可概括为：

```text
wafer -> cleaved bar -> facet cleaning / coating -> diced chip -> chip test / packaging
```

解理同时是机械分割和光学端面形成步骤。端面越平整、越干净，散射、局部吸收和膜层缺陷越容易控制；端面存在崩边、裂纹、污染或氧化时，后续光功率和可靠性会受到影响。

### 7.1 做什么：端面形成

InP 是晶体材料，沿特定晶面可以形成较平整的解理面。对边发射激光器来说，这个平整面既是光离开芯片的窗口，也是腔体边界的一部分。

Fabry-Perot 激光器主要依靠两个端面的镜面反射形成往返反馈，因此腔长和端面反射率会直接影响纵模间隔、阈值和输出功率。DFB 激光器的主要波长选择和反馈来自沿传播方向分布的光栅，中心波长首先由光栅周期和有效折射率决定；端面仍会通过残余反射、输出耦合和边界相位影响 SMSR、阈值、输出方向和器件间一致性。

二者的关系是：DFB 光栅决定哪些波长更容易振荡，端面决定这个光场在芯片边界处如何离开、反射或受到扰动。

### 7.2 怎么做：腔长与端面相位

解理位置首先决定芯片长度，也决定增益区和光栅区的实际长度。腔长会影响阈值电流、输出功率、串联电阻、热阻和端面光功率密度。高功率 CW DFB 往往会采用较长的光学腔来缓解热密度和光功率密度的压力；高速调制或小尺寸器件则会受到电容、带宽和芯片面积约束，因此腔长需要随器件目标共同设计。

对 DFB 激光器来说，还需考虑光栅耦合系数与长度的乘积 `kappa L`。这个量会影响分布式反馈强度、模式选择和边模抑制。解理位置相对于光栅周期的位置，也会改变端面处的相位边界；当残余端面反射、AR coating 不完全或两端镀膜不对称时，同一版图上的器件可能出现阈值、SMSR 和输出方向差异。

### 7.2 怎么做：端面处理与镀膜

端面处理通常包括清洁、钝化、镀膜和检验。清洁用于去除解理后的颗粒、残留物、氧化物和有机污染；钝化用于减少端面表面态、降低吸收和后续退化风险；镀膜则通过介质膜设定端面反射率，并保护高光功率密度区域。

AR coating 是 anti-reflection coating，也就是低反射膜。它可降低端面残余反馈，减弱 DFB 激光器中的残余 Fabry-Perot 效应和外部反馈敏感性。HR coating 是 high-reflection coating，也就是高反射膜。它可提高某一端反射率，让更多光从另一端输出，并改变腔内功率分布。实际器件可以采用一端 AR、一端 HR，也可以采用低反射双端面或其他反射率组合，取决于腔型、功率、耦合方式和稳定性要求。

端面镀膜同时兼具保护和腔体边界设计的功能，相当于重新设定激光腔边界上的反射率、相位和出光方向。

### 7.3 对器件的影响

端面位于高光功率密度区域，容易成为可靠性的薄弱环节。解理缺陷、污染、膜层针孔、膜厚不均、氧化或吸收中心，会将一部分光功率转化为局部热；局部温升又会增强吸收，使输出功率下降、阈值上升，并加速端面和材料退化。严重时，器件可能出现 <TermNote label="COD" note="catastrophic optical damage，灾变性光学损伤。端面或腔内局部吸收引发热失控时，激光器输出会突然劣化甚至失效。" />。

| 端面相关项 | 主要影响 |
|---|---|
| 解理质量 | 决定端面平整度、平行度、散射损耗和局部缺陷风险。 |
| 腔长 / 解理位置 | 影响阈值、热阻、端面功率密度、`kappa L` 和 DFB 端面相位。 |
| AR / HR coating | 控制反射率、输出耦合、腔内功率分布和外部反馈敏感性。 |
| 端面清洁 / 钝化 | 降低污染、表面态、氧化和局部吸收带来的退化风险。 |
| facet reliability | 影响老化、COD 风险和长期输出功率保持率。 |

端面处理完成后，器件是否真的达到设计目标，还需要通过测试将这些隐藏差异暴露出来。

### 7.4 量产壁垒

解理与端面工程的壁垒，集中在如何稳定地形成光学边界。解理、清洁、钝化和镀膜属于后段工艺，同时会直接改变端面反射、端面相位、输出耦合、端面吸收和高功率可靠性。对 DFB 激光器来说，端面残余反射和端面相位还会影响 SMSR、模式稳定性和器件间一致性。

成熟能力通常体现在 bar-level handling、解理精度、端面洁净度、膜层厚度均匀性、AR / HR coating 反射率控制、膜层应力和端面可靠性筛选之间的配合。端面工艺的难点在于：缺陷尺寸虽小，影响却极为显著。一个崩边、针孔、污染点或局部吸收中心，可能在初始测试中只表现为轻微偏差，却在高温、高功率和长时间工作后演变为输出退化或 COD 风险。因此，端面工程既是光学设计能力，也是量产良率和寿命控制能力。

## 8. 测试与筛选

激光器芯片进入后续组装之前，需要证明它在目标电流、目标温度和目标波长窗口内能够提供稳定输出。测试会将前面所有制造变量转化为数据：外延厚度和组分、光栅周期、脊形宽度（ridge width）、接触电阻、绝缘开窗、解理质量和端面镀膜，最后都会反映到 I-V、L-I、光谱、温漂、噪声和老化表现中。

### 8.1 做什么

同一片晶圆、同一套版图上的芯片也会存在离散性。晶圆中心和边缘的外延厚度可能不同，光栅周期和刻蚀深度会有纳米级偏差，金属接触和绝缘开窗会带来电阻或漏电差异，解理和镀膜也会让端面反射率和端面吸收不同。

测试的作用是判断这些差异是否仍落在设计窗口内。对激光源来说，合格芯片需要同时满足目标功率、目标波长、单模稳定性、效率、噪声、温度窗口和早期可靠性要求。

### 8.2 怎么做：外观与电学检查

测试通常先排除明显坏片。显微检查会观察崩边、裂纹、污染、金属剥落、焊盘（pad）损伤、端面缺陷和镀膜异常。电学预检会查看开路、短路、反向漏电、低电流漏电、开启电压异常和串联电阻异常。这些项目可以及早暴露绝缘层问题、侧壁漏电、结缺陷、接触不良、ESD 损伤或电流阻挡结构异常。

这一层检查与第 4 节的内容直接相关：电流注入位置、欧姆接触、钝化和电流约束做得如何，最早会在 I-V 行为和漏电测试中显现。

### 8.2 怎么做：L-I-V 测试

L-I-V 测试是激光器芯片测试的核心之一。`L` 表示光输出功率（optical output power），`I` 表示驱动电流（drive current），`V` 表示二极管电压（diode voltage）。测试会从低电流扫描至目标工作电流，同时记录光功率和电压。

L-I 曲线中的阈值电流 `Ith` 反映增益、损耗和电流注入是否达到设计目标；阈值以上的斜率效率表示电流增加时光功率增长得多快；目标功率点的电流、电压和串联电阻则反映接触、电流扩散和发热情况。PCE（功率转换效率）/ WPE（电光转换效率）描述电输入功率转换为光输出功率的比例，它的峰值通常出现在某个工作区间，而不一定是最大输出功率点。高电流下如果出现扭折（kink）、滚降（rollover）或热滚降（thermal rollover），说明模式、电流扩散、发热、载流子泄漏、光学损耗或端面状态可能已经面临相关风险。

### 8.2 怎么做：光谱测试

DFB 激光器需要确认输出是否落在目标通道附近，并且是否保持稳定单模。中心波长由光栅周期、有效折射率、温度和载流子状态共同决定；SMSR 描述主模相对边模的强度差，用来判断边模抑制和单模质量。

光谱测试还会观察波长随电流和温度的变化。电流升高会带来载流子效应和结温变化，温度变化会改变材料增益峰和折射率，因此中心波长、SMSR、线宽和 RIN 都可能随工作条件变化。模式跳变、边模增强或 SMSR 在温度窗口内明显下降，往往表明 DFB 光栅设计、端面反射、光栅耦合强度、热设计或外部反馈容忍度存在问题。

### 8.2 怎么做：温度与老化筛选

长期可用性需要在温度和时间维度上验证。激光器对温度非常敏感，高低温测试会观察阈值、输出功率、斜率效率、中心波长、SMSR、RIN 和滚降余量如何变化。测试条件需要尽量明确管壳（case）、热沉（submount）或激光二极管温度，因为不同温度定义对应不同的热路径和结温估计。

老化 / 老化筛选（aging / burn-in）用来提前暴露早期失效，并观察功率衰减、阈值漂移、工作电压漂移和光谱变化。端面污染、镀膜缺陷、微裂纹、金属扩散、位错增长和散热退化，初始 L-I-V 中不一定立刻显现，但在高温、高功率或长时间工作后会表现为输出下降和失效风险上升。burn-in 提供的是筛选和退化趋势，不等同于寿命保证。

### 8.3 对器件的影响

测试结果首先用于 pass / fail：芯片是否满足功率、电压、阈值、波长、SMSR、噪声和温度窗口要求。进一步还会进入分档：按波长通道、输出功率、工作电流、效率、温度余量、RIN、老化漂移等指标，将芯片分到不同应用等级。

这些数据也会反向服务工艺。晶圆图（wafer map）、条图（bar map）和批次统计可以显示异常是否集中在晶圆边缘、某个曝光区域、某个刻蚀批次、某个解理批次或某次端面镀膜中。测试将制造过程中的微小差异转化为可量化数据，筛选和分档则决定哪些芯片可以进入封装，哪些结果要反馈给外延、光栅、波导、电极、解理和镀膜工艺。

更完整的晶圆级测试（wafer-level test）、已知合格芯片（known-good-die）、封装级测试（package-level test）和模块级老化筛选见 [Manufacturing and test](../manufacturing-test/)。

### 8.4 量产壁垒

芯片测试与筛选的壁垒，在于能否有效量化芯片间的离散性，并将测试数据反馈给工艺改进。激光器测试设备、夹具、温控、光谱仪和老化系统本身都可以买到；难点在于用稳定、可比较、吞吐量足够的测试流程，将每颗芯片的电学、光学、光谱、噪声、温度和老化表现转化为可执行的筛选规则。

真正有价值的测试体系通常包含三层能力：第一，测试项目覆盖足够全面，能够识别阈值、效率、串联电阻、波长、SMSR、RIN、滚降、模式跳变（mode hop）和早期退化；第二，分档规则能将不同芯片匹配到合适应用，避免把所有差异简单处理成报废；第三，晶圆图、条图、批次统计和失效分析能反向定位外延、光栅、刻蚀、电极、解理或镀膜问题。测试壁垒来自数据质量、统计模型、工艺反馈速度和长期积累的判断标准。

## 9. 器件评价指标

外延、波导、DFB 光栅、电流注入、端面和筛选流程，最后都会归结为一组可测指标。对 CW-DFB 光源来说，合格的芯片不仅要能发光，还要在目标电流、温度和寿命窗口内保持足够功率、稳定波长、低噪声和可预测的退化行为。

| 指标类别 | 代表参数 | 主要反映什么 | 会影响后续哪一步 |
|---|---|---|---|
| 发光能力 | threshold current `Ith` / output power `Pout` / slope efficiency / rollover | 增益、损耗、电流注入和端面输出耦合是否匹配。 | 光功率预算、驱动电流范围和散热压力。 |
| 波长、噪声与模式 | center wavelength / SMSR / RIN / linewidth / wavelength drift / mode hop | DFB 光栅、有效折射率、温度和载流子状态是否将器件稳定在目标波长附近，并保持低强度噪声和单模输出。 | WDM 通道匹配、系统噪声、温控策略和反馈容忍度。 |
| 电学与热学 | forward voltage / series resistance / PCE 或 WPE / thermal resistance / thermal rollover | 欧姆接触、电流扩散、焦耳热、电光转换效率和热路径是否面临相关风险。 | 驱动设计、过流保护、热沉和 TEC 需求。 |
| 光束与耦合 | fast-axis / slow-axis divergence / far-field pattern / polarization / facet coating | 光场形状、端面质量和出射方向是否适合后续准直与耦合。 | 透镜选择、光纤或 PIC 耦合、偏振管理和返回光控制。 |
| 可靠性 | burn-in drift / threshold drift / voltage drift / failure mode distribution / COD risk | 端面、金属、缺陷、热应力和材料退化是否会在长期工作中放大；burn-in 用于筛选和暴露早期失效。 | KGD 筛选、封装前分档和失效分析。 |

这些指标之间并不独立。输出功率提高后，端面光功率密度和结温可能上升；温度上升又会改变增益峰、折射率和中心波长；串联电阻偏高会将更多电输入转化为热量，进一步压缩功率和寿命余量。因此，器件评价需要综合考虑功率、波长、噪声、效率、温度和可靠性窗口。

## 10. 从芯片到系统

### 10.1 封装接口

芯片测试结束后，合格裸芯片（die）将一组边界条件提供给封装设计。封装设计需要根据裸芯片的光学、电学、热学和可靠性数据，确定耦合路径、驱动窗口、散热结构和筛选流程。光源放在光引擎内部、封装边缘或前面板外置模块中，属于封装与系统布局问题，详见 [Package integration](../packaging/)。

| 接口类别 | 传给封装的主要参数 | 封装侧需要处理的问题 |
|---|---|---|
| 光学参数 | 输出功率、中心波长、SMSR、RIN、发散角、远场光斑、偏振方向、端面镀膜类型、反馈容忍度 | 透镜如何选择，光纤或 PIC 如何对准，耦合效率能达到多少，返回光是否需要额外抑制。 |
| 电学参数 | 阈值电流、额定工作电流、最大允许电流、工作电压、串联电阻、ESD 敏感性 | 驱动电流范围、软启动、过流保护、电压裕量和监控电路如何设定。 |
| 热学参数 | 功耗、PCE / WPE、热阻、波长温度系数、功率温度曲线、最高允许结温 | 是否需要 TEC，热沉和贴装材料如何选择，封装底座热阻需要控制到什么水平。 |
| 可靠性参数 | burn-in 结果、参数漂移、失效模式分布、端面和镀膜风险、批次和 wafer map 数据 | 哪些裸芯片可以进入贴装，哪些批次需要加严筛选，后续组装良率和返修风险如何估计。 |

对 DFB 激光器来说，封装热设计同时承担功率稳定和波长稳定两项任务。芯片温度变化会让中心波长漂移，也会改变阈值、斜率效率和输出功率。对 CW-DFB 光源来说，封装还要尽量减少返回光、热点和长期应力，因为这些因素会影响 RIN、SMSR、功率退化和寿命风险。

经过测试和筛选后，封装前通常希望拿到 known-good-die（KGD），也就是已经通过基本电学、光学和早期可靠性筛选的合格裸芯片。封装、耦合、封帽和模块测试的成本通常高于裸芯片筛选成本，因此在进入贴装之前先排除异常裸芯片，可以减少后续损失。

### 10.2 CPO 与系统约束

CPO（co-packaged optics，共封装光学）将光引擎置于 <TermNote label="switch ASIC" note="数据中心交换机中的网络交换芯片，负责在端口之间转发数据流；CPO 语境下，光引擎靠近它以缩短高速电互连路径。" /> 附近，缩短高速电信号路径。这个系统位置不会改变半导体激光器的基本发光原理，却会将激光源纳入更严格的热、功耗、供光路径和维护边界约束：

- 硅光 / PIC 主要负责传输、路由、调制和复用光，连续波通常由 III-V 激光器提供。
- 光引擎靠近高功耗芯片后，激光源需要在更高的局部温度、更严格的功率预算和更严的长期可靠性约束下保持稳定输出。
- 一束连续波进入 PIC 后会被分光、调制、复用并经历耦合损耗，激光器需要提供足够的输出功率和低噪声。
- 高密度光引擎要求多颗光源在波长、阈值、效率和老化速度上保持一致，系统校准、筛选和冗余设计都会受到这些差异影响。

因此，用于 CPO 的激光源需要在输出功率、热稳定性、噪声、波长一致性和寿命裕量上满足更严格的系统约束。激光器从材料、外延和芯片工艺中如何形成，决定了后续封装与系统架构能够获得多少可用余量。

### 10.3 收束

经过测试和筛选后的合格裸芯片，将一组可量化的边界条件交给封装和系统设计。激光源进入光引擎后还要面对耦合效率、热漂移、反射容忍度和长期老化等系统级命题。制造侧的责任到此为止，后续属于封装与系统集成。

## 11. 总结

一颗 InP 激光源从高纯 InP 原料和 InP 衬底开始。衬底提供晶格模板；外延定义量子阱、包层和增益材料；光刻、刻蚀和再生长定义波导、DFB 光栅和电流通道；金属、绝缘和电流窗口决定电流如何进入有源区；解理和端面镀膜定义激光腔边界和出光端面；测试和筛选将每颗芯片的差异转化为可量化指标。最后，合格裸芯片才会进入封装，成为系统中可用的连续波光源。

激光器看起来只是一颗能够发光的芯片，但它的输出波长、功率、效率、噪声和寿命，都是材料、微纳加工、端面工程、测试筛选和封装设计共同决定的结果。
