## What This Flow Does

Manufacturing and test 不是一个物理组件。它是把 CPO 从 demo 变成可交付产品的验证链路。

测试贯穿：

- substrate and epi inspection。
- wafer-level optical/electrical test。
- known-good-die。
- die attach and assembly inspection。
- optical coupling test。
- package-level test。
- link-level BER / eye / TDECQ。
- yield analysis。

### 晶圆级测试与已知合格芯片 (KGD)

在可插拔模块中，光源、PIC 和电芯片通常被封装在可替换模块内，失效后的更换边界比较清晰。CPO 把多个光引擎与高价值 switch ASIC 放到更紧密的封装生态中，单个裸 die 的早期失效会被放大成封装级良率和返修问题。

这使 **KGD (Known-Good-Die，已知合格芯片)** 筛选成为 CPO 量产中的关键质量门：

- **晶圆级测试（Wafer-Level Test）**：在解理、贴装和封装之前，尽量通过电探针、自动对准光纤和光谱测试识别不合格发光单元，并建立 wafer map。
- **老化与筛选（Burn-in & Screening）**：激光器裸芯片需要经历高温、高电流或高功率条件下的加速筛选，提前暴露早期失效风险。
- **良率计算模型（Yield Model）**：
  如果一个 CPO 系统集成了 `N` 颗激光源，单颗激光器的 KGD 良率为 `Y_laser`，那么仅由激光器引起的系统制造良率 `Y_sys` 可以粗略写成 `Y_sys = (Y_laser)^N`。
  当 `N = 16` 且 `Y_laser = 95%` 时，仅由这 16 颗光源贡献的良率约为 `44%`；当 `Y_laser = 99.9%` 时，对应良率约为 `98.4%`。这个简单模型说明，多 die 集成会显著放大单颗器件良率差异。

## Common Boundaries

- Test is not only module final test.
- Wafer-level test is harder in photonics because optical coupling itself is part of the measurement.
- Known-good-die becomes more valuable as package integration gets more expensive.

## Short Version

Manufacturing and test is the quality gate across every layer. It belongs on the homepage, but as an engineering flow, not as a physical CPO block.
