## One-Sentence Answer

光子封装难，是因为微小耦合误差、热漂移、测试时间和可靠性都会直接变成链路预算、良率和成本问题。

## Engineering Implementation

封装需要同时处理 die attach、wire bonding 或 flip-chip、fiber array、lens、spot-size converter、edge coupler 或 grating coupler、thermal path 和监控测试。

## Key Metrics

- coupling loss
- alignment tolerance
- thermal drift
- receiver sensitivity
- BER
- TDECQ
- yield

## Common Misconceptions

- 能在实验室出光，不等于能量产。
- active alignment 不是简单装配动作，而是成本、吞吐和良率问题。
