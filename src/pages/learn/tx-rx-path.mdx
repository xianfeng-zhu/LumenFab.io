## One-Sentence Answer

Tx 把电信号变成被调制的光，Rx 把光信号变回可判决的电信号。

## Engineering Implementation

典型 Tx 路径包括 DSP 或 CDR、driver、laser 或 modulator、coupling、fiber。典型 Rx 路径包括 fiber、photodiode、TIA、CDR 或 DSP、电输出。

## Terms

- Tx: Transmitter，发射端。
- Rx: Receiver，接收端。
- Lane: 一条并行承载数据的物理或逻辑通道。
- TIA: Transimpedance Amplifier，跨阻放大器。
